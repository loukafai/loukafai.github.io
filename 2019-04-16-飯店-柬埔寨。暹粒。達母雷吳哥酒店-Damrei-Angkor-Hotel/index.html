[飯店] 柬埔寨。暹粒。達母雷吳哥酒店 Damrei Angkor Hotel

2019-04-16
2018-Cambodia-6days, 2018-柬埔寨, Angkor, Cambodia, hotel, http://schemas.google.com/blogger/2008/kind#post, life, lift, trip, 吳哥窟, 旅行, 柬埔寨, 生活, 飯店

這次的【2018 柬埔寨吳哥窟六天遊】住了兩間不同的飯店，一家是便宜的「達母雷吳哥酒店 Damrei Angkor Hotel」和一家稍為高級的「梅塔 Spa 公寓 Metta Residence & spa」。

這篇是分享「達母雷吳哥酒店 Damrei Angkor Hotel」，價位便宜但仍然有不錯品質。

在 google 用「便宜」這個關鍵字找到這家「達母雷吳哥酒店 Damrei Angkor Hotel」，再找了幾篇網友的心得都分享這家飯店很新，早餐好吃，位置方便，反正就是 C/P 值（性價比）很高啦！所以便選擇了這家，住過之後都覺得可以推薦一下。

本來有考慮過的飯店有「Shinta mani resort」、「Grand Bayon Siem Reap Hotel」、「Victoria Angkor Resort & Spa」。

飯店接送

飯店是在 Agoda 訂的，有包含接送服務，不過好像只有接機或送機「二選一」，值得讚的一點是在我用 email 寫信給飯店預計接送的時候，飯店人員回覆超級快！！才不到二十分鐘就給我回信確認了。

▲ 來接機的除了司機還有一個小哥，會講一點中文，很有禮貌

▲ 車上同時有準備冰水，之後會發現柬埔寨好喜歡用冰水招待客人

▲ 入口放了個迷你版小吳哥模型

▲ 飯店有按摩服務可以訂購

▲ 即使事先沒有計劃，也可以直接跟飯店買行程

房間

▲ 床褥很軟沒錯，不過我們是比較喜歡睡硬一點的床

▲ 房間蠻大，乾淨，廁所跟浴室是分開的設計

▲ 東南亞飯店必備的「花式」毛巾，這次來到暹粒得到的是一朵蓮花

▲ 小陽台是向外街

飯店環境

▲ 飯店的小吳哥模型晚上會發光的呢！

餐廳

達母雷吳哥酒店 Damrei Angkor Hotel 吃早餐的地方是個餐廳，如果真的懶得出門覓食就可以直接在這邊吃，只是消費當然比市區貴一些些啦！

早餐

▲ 可以點到現做的湯麵

▲ 只要是出外旅行有飯店早餐的話，我都會特別多吃生菜沙拉，有時候更會加杯優格幫助腸胃暢通

▲ 我們的行程都是七點左右開始，所以六點半就要起床吃早餐了，飯店現做的柬埔寨湯麵和歐姆蛋，都很喜歡。

▲ 早上來一碗熱騰騰的湯麵馬上醒過來

▲ 享用早餐的位置。達母雷吳哥酒店 Damrei Angkor Hotel 本來就有一家咖啡廳，可以直接在這裡吃飯，蠻方便的

2018 柬埔寨吳哥窟六天遊

行程備忘

第一天行程

桃園機場 → 坐越南航空 → 轉機：越南河內 - 內排國際機場 (HAN) → 午餐：機場 牛肉河粉、法式麵法 → 柬埔寨 - 暹粒吳哥國際機場 （REP） → 手機 SIM 卡：metfone 4G → 飯店接送：Damrei Angkor Hotel → 飯店：暹粒 達母雷吳哥酒店 Damrei Angkor Hotel → Ya Tep 女神像 → Lucky Mall → Lilypop Restaurant → 吳哥夜市 Angkor Night Market

第二天行程

早餐：達母雷吳哥酒店 Damrei Angkor Hotel → 大圈行程出發：Stop Asia → 購買吳哥窟三日票 → 大吳哥城南門 (THE SOUTH GATE OF ANGKOR THOM) → 聖劍寺 （PREAH KHAN） → 涅盤寺（龍蟠寺 NEAK PEAN） → 將軍廟（達松寺 TA SOM） → 東美蓬寺（東梅蓬寺 EAST MEBON） → 變身塔（PRE RUP） → 午餐：Mie Cafe → 荳蔻寺（喀拉凡寺 PRASAT KRAVAN） → 班蒂喀黛 （斑黛喀蒂寺 BANTEAY KDEI） → 皇家浴池 （SRAH SRANG） → 巴肯山 （PHNOM BAKHENG） → 晚餐：Khmer Kitchen Restaurant → Pub Street

第三天行程

早餐：達母雷吳哥酒店 Damrei Angkor Hotel → 小圈行程出發：Stop Asia → 小吳哥 （Angkor Wat） → 午餐：Ly Ly Restaurant（麗麗牛肉麵） → 巴芳寺 （BAPHUON） → 巴揚寺（巴戎寺 BAYON） → 鬥象台 （TERRACE OF THE ELEPHANTS） → 空中宮殿 → 十二生肖塔 → 塔普倫寺 （TA PROHM） → Lucky Mall → 晚餐：Lucky Burger

第四天行程

早餐：達母雷吳哥酒店 Damrei Angkor Hotel → 午餐：Cambodian Traditional Chef → 飯店：梅塔 Spa 公寓 Metta Residence & spa → 點心：飯店傍晚 Happy Hour → 吳哥的微笑文化表演秀 → 宵夜：路邊炒麵

第五天行程

早餐：梅塔 Spa 公寓 Metta Residence & spa → 外圈行程出發：Stop Asia → 女皇宮（BANTEAY SREI） → 崩密列 （BENG MEALEA） → 午餐：貢開金字塔景點餐廳 → 貢開金字塔（KOH KER） → 點心：竹筒飯 → Pub Street

第六天行程

早餐：梅塔 Spa 公寓 Metta Residence & spa → 午餐：HAVEN → 沙樂市場（Phsar Leu Thom Thmey） → 下午茶：Temple Coffee n Bakery → 柬埔寨 - 暹粒吳哥國際機場 （REP） → 轉機：越南 胡志明 - 新山一國際機場(SGN)
