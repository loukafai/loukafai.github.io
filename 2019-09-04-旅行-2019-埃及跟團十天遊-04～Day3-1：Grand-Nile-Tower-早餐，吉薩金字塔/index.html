[旅行] 2019 埃及跟團十天遊 04～Day3-1：Grand Nile Tower 早餐，吉薩金字塔

2019-09-04
2019-egypt-10days, 2019-埃及, egypt, lift, trip, 埃及, 旅行, 生活, 非洲

【2019 埃及跟團十天遊】來到第三天。早上在「飯店 Grand Nile Tower」吃過早餐後出發，今天的景點是行程裡最最最重要的吉薩金字塔區和人面獅身像。

「人怕時間，時間怕金字塔」，四千多年前建造的字金塔，到現在仍然屹立不倒，任何東西放在在它旁邊都十分年青又渺小。

第三天行程

飯店早餐：Grand Nile Tower → 吉薩金字塔 → 金字塔最佳拍照點 → 獅身人面像 → 午餐：絲綢之路 → 哈利利市集 → 旅館盥洗 → 吉薩火車站 → 夜宿火車

早上起床，清晨的尼羅河，景色是多麼的優美宜人。

▲ 早上 06：22 而已都已經是一堆人，都是早上趕出發的團客像我們。還好我們時間充裕很多，慢慢吃沒關係。

▲ 出發去金字塔時，導遊在車上給大家的注意事項，就是「千萬不要理任何人」XD ( via 20190402 出發去金字塔 - YouTube)

吉薩金字塔

▲ 早上九時左右，我們來到金字塔了，巨大的金字塔我們在遠方就能看到

▲ 憑票入場，別看太陽那麼猛，其實才二十多度，早點來逛就不用很熱，不過墨鏡要帶好

▲ 烈日當空之下，導遊仍然花時間給大家介紹金字塔的來源和作用，金字塔其實跟大家想像中的不一樣 ( via 20190402 金字塔介紹 - YouTube )

吉薩金字塔群，是指一大片位在埃及開羅郊區的吉薩高原內之陵墓群，於1979年登入聯合國教科文組織世界遺產，於2007年7月7日獲選為世界新七大奇蹟中的榮譽奇蹟。陵墓群建於埃及第四王朝，主要由三個金字塔組成，而當中最大的是胡夫金字塔（又稱「大金字塔」），同時也是古代世界七大奇蹟中最古老及唯一尚存的建築物。 ( via 吉薩金字塔群 - 維基百科，自由的百科全書)

▲ 聽說每個金字塔裡面都差不多，導遊就選了卡夫拉金字塔讓大家進去參觀

▲ 裡面通道小小的，大隻的老外爬得好累，我們亞洲人就很輕鬆

金字塔內不准拍照，所以我們沒有拍，但是後來有在網路上找到類似的照片，金字塔內部大概就像照片中那樣，是個小石室，裡面放著一個石棺 ( via 从来没人在金字塔里找到过法老的木乃伊，那么金字塔真的是法老的陵墓吗？ - 知乎)

金字塔最佳拍照點

在近距離看完金字塔之後，然後上車去遠一點的拍照點，為了滿足大家拍照的需求，旅行團好像都會帶去遠一點的地方，跟金字塔來一次大合照

▲ 金字塔週邊都一大片是沙漠，什麼都沒有

▲ 到處都可以買到伴手禮

▲ 這邊可以把孟卡拉、卡夫拉和胡夫三個金字塔一次入鏡

▲ 除了要拍好美的金字塔，當然還要來一張在沙漠上起飛
