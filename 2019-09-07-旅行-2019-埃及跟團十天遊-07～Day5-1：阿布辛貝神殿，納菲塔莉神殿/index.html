[旅行] 2019 埃及跟團十天遊 07～Day5-1：阿布辛貝神殿，納菲塔莉神殿

2019-09-07
2019-egypt-10days, 2019-埃及, egypt, http://schemas.google.com/blogger/2008/kind#post, lift, trip, 埃及, 旅行, 生活, 非洲

來到【2019 埃及跟團十天遊】第五天。在要來埃及之前，還以為埃及只有金字塔可以看，然後開始做 "功課" 之後才發現埃及的神廟和自然景色也很值得看。例如今天開始看的第一個【阿布辛貝雙神殿】，就是好戲的開始。

第五天行程

起床咖啡 → 阿布辛貝神殿 → 納菲塔莉神殿 → 游輪午餐：Princess Sarah II → 游輪下午茶 → 康孟波雙神殿 → 鱷魚博物館 → 游輪晚餐：Princess Sarah II

起床咖啡

▲ 今天的集合時間是早上 4 時，聽到大廳有聲音，原來是游輪貼心的提供熱茶和咖啡給大家提神，還有蛋糕 XD

▲ 還有外帶的一份早餐

▲ 人手一卡，因為今天游輪會開，所以要集齊所有人才能開船

▲ 早餐很豐富的呢，有幾個麵包和水果

▲ 早上只有 20 多度，在旅遊巴上要穿個羽絨才受得了

▲ 五點半，在沙漠上看日出，浪漫的呢

▲ 旅遊巴一直往沙漠開，是一整隊的巴士一起開，聽說是為了互相照顧，因為在沙漠中手機都沒有訊號，如果車子壞掉就很危險了

▲ 中間會停在休息站一下，上廁所每人都要 5 埃磅小費，有夠貴的

▲ 第一次近距離接近沙漠，大家都在拍照

▲ 時間是 8：25，坐了大概 3 個小時的巴士終於到達阿布辛貝神殿

▲ 在阿布辛貝神殿內拍照要另外買拍照券，300 EGP 一張，難得來到當然要拍好拍滿

阿布辛貝神殿

阿布辛貝神殿，是一處位於埃及亞斯文西南290公里的遠古文化遺址。它坐落於納賽爾湖（Lake Nasser）西岸，整座神殿是一整座岩山鑿刻而成的，被聯合國教科文組織指定為世界遺產。 由依崖鑿建的牌樓門、巨型拉美西斯二世摩崖雕像、前後柱廳及神堂等組成。1964年開始至1968年間，因興建亞斯文水壩而整體遷移至高出河床水位60餘米的後山上，是世界文物建築保護方式的成功嘗試。 ( via 阿布辛貝神殿 - 維基百科，自由的百科全書)

▲ 繼續是導遊介紹 ( via 20190404 阿布辛貝雙神殿 - YouTube)

▲ 覺得當年聯合國用了 4 年時間把整個神殿搬走的工程真的是奇蹟

參考：埃及偉大神廟的搬遷偉業 - BBC 英伦网

▲ 在阿布辛貝神殿內拍照要另外買拍照券，300 EGP 一張，難得來到當然要拍好拍滿，建議要拍最好都要買券，因為在現場看到工作人員抓的很嚴，被抓到要罰錢或給小費，很麻煩。其實出來玩主要是求開心對不對，這種還算少的錢不用太省？

參考：請問各位大大,新的拍照規定???

▲ 三千多年前的石像和壁畫，到現在還有幸看到

阿布辛貝勒神殿深達60餘公尺，最深處建有包括拉美西斯二世在內的四座神像，而在每年2月21日（傳說拉美西斯二世的生日）及10月21日（傳說拉美西斯二世的登基日）這兩天，陽光會直接照進洞內最深處，而且會神奇的只照射在拉美西斯二世的神像身上。 (via 阿布辛貝神殿 - 維基百科，自由的百科全書)

納菲塔莉神殿

▲ 另外一邊就是拉美西斯二世為皇后妮菲塔莉所建的神殿

妮菲塔莉（約公元前1290年—前1254年）是古埃及新王國時期第十九王朝法老拉美西斯二世的大王后。「妮菲塔莉」這名字有「美麗的伴侶」、「最美的女人」或「最好的女人」的意思，一般認為她是拉美西斯二世最寵愛的王后。 ( via 妮菲塔莉 - 維基百科，自由的百科全書)

▲ 壁畫上都是一些拉美西斯二與皇后妮菲塔莉的故事

▲ 這幅畫是講拉美西斯二世很捨不得妮菲塔莉去世，在紀念她

▲ 逛到 10 點左右就要集合離開了，同樣是有伴手禮時間
