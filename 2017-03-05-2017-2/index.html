2017 年 2 月影視清單，個人觀影後的排名如下：

  1. 逍遥法外 第一季 / How to Get Away with Murder Season 1
  2. 天空之眼 Eye in the Sky (2015)

逍遥法外 第一季 / How to Get Away with Murder Season 1 / 谋杀入门课(台) / 天才刑法班

via 逍遥法外 第一季 (豆瓣)

一部人氣美劇，我用了 2 個多月才看完，因為劇情很精彩所以捨不得一次看完。

劇名“How to Get Away with Murder”，所以劇情主要是是關於謀殺案。女主角是刑事律師兼大學講師，她在大學任教的班上招了幾個學生當她律師事務所的實習生，好死不死就接著發生了一件命案，劇情就開始展開。

本季的劇情以男主角鄰居牽涉的命案為主軸，講一個女大學生莫名其妙被殺後丟到水箱裡。每集開頭都以一小片段作為倒敘，過程充滿懸念，就是為了吸引大家繼續追看啦。而除了懸疑情節，劇中更是滿滿的「肉搏」戲份，不止是男女，甚至連男男都有，一些角色竟然會意想不到的睡在一塊 Orz，床戲太多，有時候拍得太露骨的部份我都會跳過，因為真的會受不了 >///<

第一季的結局實在令人意想不到，個人覺得有點勉強，因為這樣的話 Sam 真的死得很冤枉呀 XD

天空之眼 Eye in the Sky (2015)

via 天空之眼 (豆瓣)

這戲是講男主角負責操控無人偵察機去執行刺殺恐怖份子的任務，過程中男主角為了不讓賣麵包小女孩受傷而用盡辦法，甚至差點讓線人送命。電影裡可以看到有人會為了達到目的不擇手段犧牲別人、可以看到官僚主義是就是相互推卸責任，因為根本沒有人願意出來承擔。

如果是你，會為了拯救 80 人而犧牲 1 個無辜小女孩嗎？在扣下發射飛彈的板機前，你會不會猶豫？

然後這部電影的好看之處是節奏相當明快，一氣呵成絕無冷場，值得一看。

完美陌生人 Perfetti sconosciuti (2016)

via 完美陌生人 (豆瓣)

一部頗具話題的法國電影，劇情故事簡單、製作成本很低，吸引之處在哪？也許是能夠引起大家的共鳴。

現代人的手機裡藏了大量的個人隱私與秘密，所以劇情就是講他們發起一個遊戲，在聚會全程把手機收到的訊息和來電跟大家一起分享，結果當然是引爆出大量的問題。

隨著劇情推進，每位角色都被發現了一些「不能說的秘密」，個人比較深刻有二段：第一段是女兒打給爸爸問要不要去男友的家過夜（發生初夜），然後爸爸很開放地講解“要不要發生都是你的自由，但是如果你有猶豫或不想要的話，你還有時間可以等待”；第二段是一位角色因為不想被發現是男同志，所以盡力去隱瞞。

雖然表面上鬧得很大，但是到最後大家離開時好像什麼都沒有發生過一樣，外國人就是比較開放的樣子。

奇異博士 Doctor Strange (2016)

via 奇異博士 (豆瓣)

原來是不太看好這部，但看完結果是比想像中要精彩很多，因為畫面特效製作相當到位，劇情對白充滿幽默感，唯一不太滿意是最後 Boss 好弱，高潮看得一段不夠過癮。

近年的超級英雄片一直推出，單一位主角已經不能滿足觀眾的欲望，所以最期待是看到多位英雄出現在同一部電影裡，看 2016 年最賣座的電影是《美國隊長 3》就知道啦！好想看到紅女巫大戰奇異博士哦～～

無恥之徒(美版) 第二季 Shameless Season 2 (2012)

via 無恥之徒(美版) 第二季 (豆瓣)

算是前陣子比較愛看的美劇，喜歡的是這劇的對白比較生活化，可以順便學學英文，然後就是看主角一家在很窮又缺錢的情況下如何掙扎生存，帶有一點點勵志的 fu

這季的重點角色是 Philip，因為他原本準備要當爸爸了，結果被戴了一頂大大的綠帽，打擊好大。他這個角色十分矛盾，既然是理科天才但又不願意去好好唸書，因為他覺得就算賺到錢也只是被姐姐利用作為生財工具，所以情願做一些非法的勾當去討生活費。希望這個角色之後會有突破性的發展，不過我覺得沒有可能，因為角色有缺點才會吸引人繼續去看追看。

《無恥之徒》是一部有空可以看看，主要是用來調節生活壓力的美劇。笑點不多，但看劇中角色一直在耍賤招，想盡辦法用力討生活也是蠻爽的。

降臨 Arrival (2016)

via 降臨 (豆瓣)

男主角不再是鷹眼人，女主角也並非超人的女友。

以為是一部外星人大戰電影，但原來是部蠻文藝風格的科幻片。看電影的前半部分，那種組織一隊人前往未知文明開發的感覺，讓我想起早幾年的《普羅米修斯 Prometheus》，而後半部分，女主角從外星文字中習得預知能力，從而改變了未來的戲碼，想起了《星际穿越 Interstellar》。

這是一部比較有深度的戲，評價好壞參半，如果希望看到人類與外星人大戰的戲份，一定會感到失望，因為半點都沒有。但如果是喜歡研究電影的朋友，劇情所想要探討的主題內容應該足夠帶給觀眾很多的思考。

這部《降临》在豆瓣有很多心得，對電影有興趣一定或有看沒有懂的人可以去看一下人家寫的影評，就會發現這部戲裡要探討的道理真的蠻多。

跑吧愛情 (2015)

via 落跑吧爱情 (豆瓣)

一部集多功能的陸台合拍片，既能把兩位主角演員（任賢齊、舒琪）保持曝光率，推銷澎湖觀光，又暗地裡把大陸綜藝的相親節目、淘寶網推了一把。

整部戲都合符水準和期待，不過不失。但比較不懂的是戲名，也許《落跑吧愛情》是指劇情尾段纷纷（舒淇 飾演）為了幫助家族生意而離開澎湖，就這樣"落跑"了？但為何又要在新書發布會上講在還在等男主角去遵守約定呢？

所以我覺得戲名應該要改為《奔跑吧愛情》比較貼切，還可以順便推廣《奔跑吧兄弟》呢 XD

最深刻一幕是男主為了女主潛水去找行李，還因此而受傷送院，女主瞬間被感動，懂了嗎？男生只要做到了一件讓女生感動的事，就夠了。
