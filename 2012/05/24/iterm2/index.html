▲ 近來把 iTerm2 配 vim 來取代原本在用的 MacVim

● 原因

  1. 快捷：不用時掛在背景，使用時透過全域鍵快速啟動 iTerm2
  2. 方便：不像 Macvim，因為 iTerm2 原生就是一個 Terminal，在需要用到 Terminal 做事時會更方便
  3. 漂亮：用上了 Solarized 的配色和語法著色，再加上一點半透，很好看

● 使用心得

▲ 設定「CMD + \」全域快速啟動 iTerm2

▲ 字型設定用 " Monaco " 大小 16

▲ vim 使用 Solarized 的 colorscheme－sol-vim，iTerm2 用 Solarized Dark.itermcolors 配色

● Vim 裡的文字複製

因為 iTerm2 不像 MacVim 可以在 visual mode 直接用 CMD＋c 複製文字到系統剪貼板，所以我會在 .vimrc 加入這句，然後就可以在 iTerm 裡的 Visual mode 選取文字後用 <Ctrl＋c> 複製

vmap <C-c> y:call system("pbcopy", getreg("\""))<CR>
