▲ LPI Level-1 考過了

自從考完 CCNP 之後就很對考證照這東西上癮了，於是就繼續看看有什麼可以考又不太難的。

剛好看到有 LPIC 這個關於 Linux 的證照可以試試（LPIC 是什麼？），LPIC 有三級，Level-1、2、3，原本想直接考 Level-2 就好，不過原來要先有 Level-1 的資格才可以拿到 Level-2 的證書。

因為自己對於 Linux 和架伺服器有些興趣，可以借考 LPI 迫自己加強對 Linux 的功力，順便也為自己的 CV 加點分。

▲ Level-1 共要考兩科 117-101 和 117-102，去 VUE 考就可以了，每科 150 美金

考試形式是電腦作答，題型有 1) 選擇題 2) 指令題 一共 60 題，90 分鐘時間，說真其實難度頗大，所以一般需要考古題幫忙一下。
