Samsung Galaxy Note 新手機入手，分享一下使用心得

● 入手原因

▲ 我的上一台手機是 Apple iPhone 3GS，雖然 iPhone 真的是簡單好用，可是它有些地方是我不能滿足我：

  1. 螢幕小（3.5 吋）
  2. 看網頁不會自動排版
  3. 沒有閃光燈
  4. 沒原生支援 asf 檔

加上我每天上下班都需要通勤（約 1 小時），在公車上最適合用來打發時間的事就是看新聞、看fb、看資料，每天都利用一下這樣的時間多做一點事，讓人生充實更多

所以既然我有這樣的需求，那麼選一台螢幕大一點的手機也不過分吧 XD

● iOS 與 Android 差很多？

▲ 因為習慣用 iPhone 久了，都直接把 App 丟在桌面，也沒有特別放什麼小工具

其實會玩手機的人玩 iOS 或 Android 都一樣，不過在整體上 iOS 還是比較直覺，App 的質素和數量也較好，如果 iPhone 未來推出 4.5 吋（含）以上版本的話，我會想轉回去 iOS 懷抱

● Galaxy Note 會怕太大台嗎？

▲ SGPCASE 手機套，黑色隱若看到閃閃粉，防滑性很好，對於一支大尺寸的手機很有幫助

我想很多人在入手這支 Note 前都尺寸上的考量，以我自己常穿牛仔褲的感覺，大概只是偶爾彎下腰會卡到，平常拿在手也可以握得住，打字（用倉頡）用單手也 OK，看在它有 5.3 吋螢幕的份上，尺寸問題絕對可以接受

● 使用心得

  1. 速度快、螢幕大，看網頁會自動排版很舒服，推薦剛好想在平板或手機上做抉擇的朋友直接選這支
  2. 用手背橫掃螢幕可以快速截圖，超喜歡這功能
  3. 「分享」功能做得很方便，網頁分享到 Whatsapp、照片分享到 Flickr 都很快又直接
  4. ROOT 機
  5. Root 後可以弄拍照快門靜音
  6. Root 後可以弄在通知列加入數據開關
