▲ 威記倒數

在 2012 的最後一天意外地去了威記除夕倒數，這雖然是原先是沒有想過會進行的活動，不過說真的，威記的氣氛真的很好，很多外國人（包含自由行 XD），酒店同時也在現場放了一隊 band 都在唱外語歌，而且表演者還會把現場觀眾氣氛帶動起來。

要說重點，就應該是在踏入 2013 00：00 時，威記還有附近幾家酒店一連串燒起了煙花，超浪漫的。

新的一年，新的挑戰。

——————

▲ 賣了 Note II

因為手機上台買了這部 Note II，不過沒有要用的打算，於是在 cyber 上 4900 賣掉了，算是便宜了些預繳。

說真的其實很想留下 Note II 賣了現在用的 Note，不過舊的用的還可以而且不想多花錢，還有一個原因是 Note 用久了會覺得太大台不方便，下一台應該會選小支一點的。

——————

▲ Note 脹電

買 Note II 的時候剛好發現我那台 Note 的電池脹大了，雖然很想順便換機，不過還是放棄了，去買了一塊新的繼續用 MOP 140，算是可以接受（如果這塊是真正原厰電的話 XD）。

朋友說，電池脹大了是因為：

  1. 長期一邊充電一邊用手機（所以他都不用尿袋）
  2. 每次都等到電量只剩很小（15％ 以下）才去充電

回想一下我確是常常發生上述的情況，之後要注意一下。
