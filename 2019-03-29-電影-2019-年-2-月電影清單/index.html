2019 年 2 月影視清單，個人觀影後的排名如下：

  1. 無雙 (2018)
  2. 麻理惠的整理秘訣 Tidying Up with Marie Kondo (2019)
  3. 神奇動物：格林德沃之罪 Fantastic Beasts: The Crimes of Grindelwald (2018)
  4. 蜘蛛俠：平行宇宙 Spider-Man: Into the Spider-Verse (2018)
  5. 大師兄 (2018)

無雙 (2018)

via 無雙 (豆瓣)

本來對這部不太注意，但是聽到影評說很精彩，當然要馬上找回來看看。

香港人拍犯罪片和警匪片一直都很拿手，《無間道》就是當中的經典作，而回顧一下之前的電影，講製作假鈔假支票的電影好像不多，近十年以來最有名的「做假」電影當然是《 Catch me, if you can 神鬼交鋒》(2002)，到之後的《盜海豪情 Ocean’s Eleven》系列都是十分棒的作品，直到看完了最近的這部《無雙》之後，刷新了我個人對犯罪電影排名的記錄。

先說結局，個人實在很欣賞這部的劇情，因為真是看到最後才知道周潤發是真是假，到底是不是郭富城幻想出來的人物，這種一直保留伏線到最後才知道結局的電影最合我口味了。

翻查一下網路，「無雙」是指獨一無二的意思，而電影劇情中間又有講到一句對白，「有的時候，假的也許比真的好。」不禁讓人反思當中的道理，這樣能夠又與電影用超高彷偽鈔作為主線劇情互相呼應。有時候喜歡一個人，到底是喜歡他的表面，還是喜歡他的內在，還是必須缺一不可？

「你知不知道能做大事的男人，都是為了女人，我爸爸跟我說過，放棄愛情的男人，什麼事都做不成。」

《無雙》另一個有趣的地方是裡面滿滿的金句，再加上郭富城和周潤發兩位男主角的演技，讓整部電影更加精彩。然後電影也有相當多的槍戰戲碼，看到發哥拿雙槍對決，完全呈現當年英雄本色的風範。

十分十分推薦給喜愛犯罪、懸疑電影的朋友看這部《無雙》。

大師兄 大師兄 (2018)

via 大師兄 (豆瓣)

會看這部完全是為了宇宙最強的甄子丹呀！！只是偏偏這部裡面的文戲比較多，動作戲只有一點點，如果是想看武打戲的朋友應該會失望。

電影是想要推廣有教無類，作育英才的想法是很值得讚賞，只是找來甄子丹來當男主角就有點不太適合，文戲部分拍得好弱，打鬥戲又被縮減，結果兩邊不到岸，實在可惜。

最後這部《大師兄》只推薦給甄子丹的鐵粉。

麻理惠的整理秘訣 Tidying Up with Marie Kondo (2019)

via 麻理惠的整理秘訣 (豆瓣)

前陣子很火紅的網劇，是教人整理家裡的雜物。

這部會在歐美地區紅起來，應該是他們家裡一般面積都很大，堆了太多的雜物，久而久之日常生活就被家裡的雜物淹沒，影響生活品質。

如果歐美地區有這樣的問題，那麼我們身處人口稠密的亞洲地區，問題一定更嚴重，所以，好幾年前就已經有「斷。捨。離」這個名詞出現，核心的思想就是要捨得丟掉無有用的東西，留下真正值得珍惜的物品。當你願意丟掉一堆東西之後，就能夠輕鬆生活，考試都能 100 分的樣子。

個人覺得整理的習慣是不能被迫出來的，有些人就是喜歡家裡亂糟糟的樣子，反而更有安全感（？），所以說，如果是對家裡的現況不滿，希望去做一些改變但是無從入手的話，再來看這部美劇會比較有效。

《麻理惠的整理秘訣》裡面的案例都是經典，先是有養小孩的父母把家弄亂，再來是有準備成為父母的想要把雜物清走後迎接小孩，又有是丈去世之後希望透過整理重新開始。

相信觀眾們都能夠在這部短美劇當中得到一些體會。

蜘蛛俠：平行宇宙 Spider-Man: Into the Spider-Verse (2018)

via 蜘蛛俠：平行宇宙 (豆瓣)

這部動晝完全是衝著蜘蛛俠才會去看，加上動畫本來就可以把動作畫面和設計弄得很誇張，充滿創作性，所以覺得這部《蜘蛛俠：平行宇宙》是頗精彩的。

這部的風格和劇情與以往的英雄電影稍有不同，沒有一直沿用英雄電影的那種「打 Boss」套路，反而有點自我鼓勵的感覺，看著男主角由初出道變成蜘蛛俠的小伙子，一開始不太接受自己的身份，到後到信心滿滿的拯教了世界，要說這是一部厲志電影也並不過份。

這部一定推薦給喜歡看動作、看特效電影的朋友。

神奇動物：格林德沃之罪 Fantastic Beasts: The Crimes of Grindelwald (2018)

via 神奇動物：格林德沃之罪 (豆瓣)

我想縱使不是哈利波特的粉絲，但是作為電影愛好者都必須要看這部大作的續集。

個人對於這集的評價是很可以啦！畫面特效都有維持到水準，然後有把「怪獸」系列必備的許多神奇怪物加進去，當中又有許多出自哈利原著的彩蛋位，完全照顧到舊有粉絲。

這集比較有感足的一段是講奎妮為了愛情執意加入黑暗勢力，雖然是很可惜，但卻是無可奈何之下的選擇。我們都希望自己能夠理性地看待愛情，但是愛情本來就不能理性地處理。受傷過、心痛過、大哭過，最終成就了我們的人生。

這部是不需要推薦就自然會看的電影。
