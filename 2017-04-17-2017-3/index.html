2017 年 3 月影視清單，個人觀影後的排名如下：

  1. A LIFE～深愛的你～
  2. 長翅膀的猪 When Pigs Have Wings
  3. 深海浩劫 Deepwater Horizon

A LIFE～深愛的你～ (2017)

via A LIFE～深愛的你～ (豆瓣)

首先，木村拓哉新劇，必看。

這部「A LIFE～深爱的你」演員陣容很強大，木村拓哉、竹内结子、松山研一、木村文乃、菜菜绪、浅野忠信（其實這位我比較不熟），光看卡士就已經是一部可以一看的日劇，而且更是以醫生故事為提材，是日劇比較善長的種類，出來的效果當然也不錯。

說劇情部份是真的很無腦，就二四六八很容易猜到的劇情，故事在發展中也沒有留下太多的懸念，所以，演員就是本劇唯一的看點。

整劇裡面說最喜歡的演員就是松山研一跟木村文乃這對假情侶，看他們「你一言我一語」的斗氣，為本劇增加一些笑點。冲田一光（木村拓哉 飾）是本劇最可憐的角色，先是被調到國外去，然後初戀情人（竹内结子）被娶走，過了十年被叫回來，又被利用作為工具人動手術，從頭到尾他都是演大好人的角色，諾貝爾和平獎都要頒一個給他了。

長翅膀的猪 When Pigs Have Wings (2011)

via 長翅膀的猪 (豆瓣)

一部有趣的巴勒斯坦電影，是講一個漁夫有天撿到一頭豬，但是伊斯蘭教人原來是很討厭豬這樣的生物，然後漁夫原來也不知道要怎樣處理這頭豬，誰想到剛好一個機會的情況下，漁夫發現這頭豬可以幫他賺到意外的收入，從此這頭豬改變了他的命運。

豬，不管在東方西方都可以成為餐桌上的美食，但同樣的一頭豬，去到了中東國家就變成人人喊殺的怪物。一樣的豬，怎麼去到了不同的地方，會被這麼大差異的對待？電影是用一個荒誕搞笑的故事來宣揚背後和平的重要。話說在拍這部戲的時候，中東的難民問題還未爆發起來呢。

深海浩劫 Deepwater Horizon (2016)

via 深海浩劫 (豆瓣)

電影是由真人真事改編，是講在一座海上鑽油台發生火災事故的經過。

看評分好高所以就看一下，特效普普，演員發揮也一般，就是男主角義勇救人的一段比較讓人感動。比較特別的心得就是不管生活上、工作上、凡是關乎人命的東西都要小心一點比較好，有時候看日本人做事很一板一眼，很嚴苛，但真的出問題時就是誰都不願意看到囉。

網聘女伴 Mike and Dave Need Wedding Dates (2016)

via 網聘女伴 (豆瓣)

很瞎的一部美國爆米花電影，想不到當年帥氣又受歡迎的 Zac Efron 會慢慢走去拍這種電影，覺得他當年的《歌舞青春 High School Musical》都白拍了。

女主角 Anna Kendrick 也是我喜歡的女演員（之一），同樣是自從《Up in the Air》之後都沒有新的代表作，當然，2012 年的《歌喉讚 Pitch Perfect》是有受到歡迎，只是紅歌不紅人。

所以，當演員真的要講運氣，有多少人可以像 Emma Stone 一樣，剛跑去演音樂電影就拿大獎，我想 2014 年的《鳥人 Birdman》是幫了她一把，讓她大大的提升了知名度，機會，總是留給有準備的人，希望 Zac Efron 和 Anna Kendrick 之後有更好的表現。

胡狼来了 자칼이 온다 (2012)

via 胡狼来了 (豆瓣)

一直藏在電腦裡的電影，一天不小心打開了，看過，還行。

韓國電影一向都不是我的菜，不過這部《胡狼来了》裡，宋智孝的表現是比較出色的；劇情也算可愛，經典的錯摸大堆頭戲，笑點不多，但足夠消耗半杯爆米花了吧。對了，電影的結局是有點出人意料，但覺得有點勉強，有種為驚喜而驚喜的感覺。

既然以男女主角為賣點的電影，一定要找些長得好看的呀，宋智孝還說可以，氣質高嘛！但對男主角比較無感，算了，看過就好。

奪い愛、冬 (2017)

via 夺爱之冬 (豆瓣)

以出軌、報復為主提的日劇，算是比較少見看到的類型，第一集的床戲是很厲害，但是越演下去越覺得奇怪，再看到男主角的黑化，女主角的要愛不愛，看得讓人咬牙切齒。

男女複雜關係的劇情，從來只有在美劇上發生才會合理，日劇圈裡提材大膽的類型也是有，但是不多，敢拍出來就要有被討厭的風險。

兩位主角，倉科加奈和三浦翔平，明明就是男帥女正，本來還以為日劇拍出來會有不錯的效果，但劇情方便就是不能讓人接受，覺得最大的問題是女主角的劇情設定，那種見一個愛一個的人（簡稱"出軌"囉）真的很討厭，口裡說不，身體卻很誠實，偷吃都被對方老婆抓到了還不死心，好吧，也許是我看得太投入了 XD
