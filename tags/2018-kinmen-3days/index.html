[旅行] 台灣金門三天遊~~Day03

金門之旅第三天。今天逛到有特色的【北山播音牆】，還有講解很精彩的【莒光樓】，最後更買了很有金門特色的蚵嗲和燒餅回家。可以不誇張的說，金門是個人認為台灣最好玩的地方，因為每處景點都充滿了歷史的痕跡，在參觀的同時會認識到過去曾經有數百萬人為了和平付出了相當多的努力，透過講解人員的口中，會了解到更多金門的故事。所以說，金門值得一來。第三天行程 退房：金門縣三元民宿 (San Yuan Guest House)    →    Read more

[旅行] 台灣金門三天遊~~Day02

金門之旅第二天。金門的景點很多，我們本來計劃只是想到每個景點去逛逛拍照，但是在【得月樓】由介紹人員聽到精彩生動的講解之後，就去找之後每個景點的講解時間去參觀。這些景點經過【金門國家公園】人員的介紹之後真的學到很多景色以外的東西。如果問我金門有什麼好玩，我會說來古蹟景點聽講解是最好玩的部份。第二天行程 早餐：永春廣東粥    →    Read more
