因為之前一直用的 QCY T1 耳機在使用了約一年之後出現左邊耳機聲音異常，剛好一年過保不能修，就去 QCY 商城找找看有沒有新產品，就很 "剛好" 的找到這支 QCY T5 XD 。

用了幾個月之後寫個使用心得。

喜歡它的優點：

  1. 便宜
  2. 配載舒適也牢固
  3. 續航力長
  4. 藍牙連接反應快
  5. 外型好看

還可以接受的缺點：

  1. 充電盒觸點不夠緊，容易放入後沒充電
  2. 不能直接控制音量
  3. 耳機觸控不夠靈敏

這支 QCY T5 是在去年雙 11 的時候入手，只要 96 RMB 好便宜就果斷入手，這個價錢只是大品牌的耳機例如：Jabra Elite 75t ( HK$1,398 )、AirPods Pro ( HK$1,875 ) 的零頭而已吧！那麼便宜你還想怎樣？

最近 (2020/2/27) 也有看到有優惠，一支 99.9 RMB
