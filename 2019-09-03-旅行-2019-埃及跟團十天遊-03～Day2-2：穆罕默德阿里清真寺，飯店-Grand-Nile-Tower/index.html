上午參觀完【埃及博物館】後，走了半天那麼累，接下來當然就是午餐時間囉！！

埃及的食物很多都是用烤的，導遊解釋說是因為沙漠國家的水很珍貴呀！反正用燒烤的方式也很好吃，我們都很習慣在埃及吃到的食物，印像最深刻的是甜點都非常非常的甜，跟日本人的伴手禮有得拼。

第二天行程

開羅機場 → 埃及博物館 → 午餐：Andrea Memphis Maadi → 穆罕默德阿里清真寺 → 薩拉丁城堡 → 晚餐：Citadel View Alain Le Notre → 飯店：Grand Nile Tower → 飯店兌換埃磅 → 飯店康樂設施

午餐：Andrea Memphis Maadi

▲ 午餐安排是"尼羅河畔埃及風味料理"，一家在尼羅河邊的船上餐廳

參考

  • Andrea Memphis Maadi，开罗Andrea Memphis Maadi点评，Andrea Memphis Maadi美食推荐 - TripAdvisor(猫途鹰)

▲ 午餐就是看著尼羅河，邊吃午餐，很浪漫有沒有？

▲ 可以點飲料，不過要自費，不過雄獅旅行社的團有保證每餐都有免費水提供

蔬菜先到，然後是幾款醬汁，都是埃及特色的東西，有鷹嘴豆泥、芝麻、茄子泥、奶酪泥，接下來的日子每餐都看到它們

▲ 然後用這些阿拉伯餅沾著吃，軟軟乾乾的，是阿拉伯特色

▲ 烤肉球

▲ 主餐是烤雞腿一大隻，雞腿用香料烤得好香，只是有點乾，沙漠國家的料理很多都是乾乾的，因為水很珍貴呀

▲ 飯後向甜點，埃及的糕餅甜得要命

穆罕默德阿里清真寺

▲ 之前去馬來西亞有參觀過很多清真寺，來到埃及發現有更多清真寺。不同的是這次多了導遊的介紹，認識多一些

▲ 導遊的穆罕默德阿里清真寺介紹 ( via 20190401 阿里清真寺 - YouTube)

▲ 這邊有個軍事博物館

來到開羅，城際的穆罕默德-阿裡清真寺高高在上、大氣顯眼——穹頂主殿高達 52 米，東側兩座附塔更在 84 米以上。在寺中游覽一番，您便知道為何它又名“雪花石之寺”。因為內、外高 11 米的城牆均用潔白的雪花石砌成，真可謂薩拉丁大城的頂上王冠。這座寺廟是在穆罕默德-阿裡巴夏（Mohamed Ali Pasha）的要求下，於 1830 年至 1848 年間由建築師 Yousef Bushnak 主持修建。 ( via 埃及旅游管理局 - 穆罕默德-阿里清真寺（Mohamed Ali Mosque）)

▲ 導遊介紹清真寺的圓頂是以前為了中間的人在講道理時把聲音更好的傳出去，是古人的智慧呀

薩拉丁城堡

薩拉丁城堡位於開羅城東郊的穆蓋塔姆山上，是11世紀時期薩拉丁為抗擊十字軍東侵而建造。位於公路主干線邊的薩拉丁城堡巍峨雄偉，城堡前面是一片綠草地，城牆寬2米，城堡分內城和外城，城內建有宮殿和寺廟，阿裡清真寺的寺頂尖塔像一把利 劍高聳入雲，巨大的圓頂沐浴在陽光下熠熠生輝。眾多的游客前去參觀拜謁和憑吊當年抵抗十字軍的英雄。 ( via 萨拉丁城堡_百度百科 )

▲ 監獄裡的蠟像也太逼真了吧

▲ 剛好聽到清真寺播放音樂，叫信徒們快點去參拜

▲ 看到對面是開羅有名的「蘇丹哈山清真寺」

▲ 來一張耶穌光！

▲ 然後又跑去另一個清真寺參觀

▲ 導遊給大家講了一些關於伊斯蘭教的東西，反正宗教都是導人向善的東西，不要當壞人

▲ 天花的燈好漂亮

▲ 埃及的車牌上主要是顯示看不懂的阿拉伯文，阿拉伯數字是小小的在下方

晚餐：Citadel View Alain Le Notre

▲ 2 個多小時前才吃完午餐，接下來又到晚餐時間，領隊知道可能太早了，所以讓大家在這裡公園逛一下，很漂亮，反正晚餐的餐廳就在這裡

參考

  • Citadel View Alain Le Notre, Cairo - Restaurant Reviews, Photos & Phone Number - TripAdvisor

▲ 正在模仿古埃及的畫來拍照

▲ 同樣是先來阿拉伯餅配特色的醬，我跟正妹都很喜歡，因為特色食物，回去之後就吃不到了

▲ 一些酸辣前菜

▲ 湯

▲ 主菜是魚肉，不像是午餐那樣乾乾的食物，團友們都很開心

飯店：Grand Nile Tower

▲ 晚上六點多，到達飯店，這家【Grand Nile Tower】是一家四星級的飯店，靠近尼羅河，景色很棒，設備齊全

參考

  • Grand Nile Tower - Google 地圖

▲ 房間大大的，窗外夜景是很美的尼羅河

▲ 亮亮的是 Sofitel 飯店

▲ 大飯店都有國際插座，第一晚不用擔心用電的問題

▲ 金色亮亮的電梯

飯店兌換埃磅

▲ 早上在機場沒有換到，還好飯店內也有開設了兌換店，匯率也不差

▲ 我們用 80 美金換到 1380 埃磅，匯率還可以，主要用來買伴手禮和上廁所

飯店康樂設施

▲ 健身房有夠大的，可惜我已經吃完飯，不然一定要跑個幾 K

▲ 還有一堆飛輪，可以來上課的樣子

▲ 戶外泳池
