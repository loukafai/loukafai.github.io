在日本 Amazon 看到想要的東西，不過他們除了書和 CD 外都不可以送貨到澳門，所以要找代購或轉送幫忙。

代購？轉送？

在澳門有好幾家做代購的店，問過朋友在哪家買日本的東西比較好，他們有推薦過一間認為不錯的 U-LUCK。

U-LUCK 的服務不錯，報價迅速（不過原來 facebook 不能報價，要寄 E-mail），而且提供一站式服務，報價後直接銀行轉帳就好了。本來想找這家本，不過總覺得他們會收商品總價的 10% 作為服務費有點貴，所以就在網路上繼續找別的方法，於是找到了 Jshopper。

==================

轉送

什麼是轉送？jshoppers 上有這樣的介紹：

將在日本國內購物網站所購買之商品，以合理的價格並安全的運送至海外郵寄地址之服務。讓您能夠在沒有提供海外寄送服務的日本國內購物網站上購買商品。

==================

過程

▲ 6月25日在 Amazon JP 刷卡付錢，6月27日送到轉送的地方

Jshopper 上午收到貨後，下午就寄 E-Mail 報價，不過他們報的重量比商品網頁上的 Shipping Weight 由 780g 增到了 1220g 所以運費比預算超出了很多。

收到報價後原本就想要馬上刷信用卡付運費，不過我的中國銀行 MasterCard 一直刷不過，但是朋友的中國銀行 Visa 一刷就過了，超不爽。不過也有可能是 Jshopper 信用卡認證寫得不好的關係。

▲ 刷卡後隔天 Jshopper 就把貨寄出了，他們會附貨件提單碼，可以在 DHL 上查詢進度，Jshopper 用 DHL 這點最滿意，因為 DHL 比 EMS 快太多了。

▲ 由付費到收到包裹一共花時 7 天，等最久的是 7月1日，因為香港回歸貨運也放假。

==================

費用

這部份應該是最重點的地方，下面做個比較，以 20,000 日元 1.5kg 的包裹來算：

  • 代購費用 ( 20,000 + 服務費 2,000 + 運費 3,000 ) x 0.0835 匯率是代購店家 = 2087 港幣
  • 轉送費用 ( 20,000 + 服務費 100 + 運費 2,280 ) x 0.083 匯率是刷卡銀行 = 1857 港幣

代購：雖然比較貴，但優點是比較簡單，也不用怕轉送的過程中會有什麼問題

轉送：優點是便宜、速度也比較快，但是需要自備信用卡，轉送過程中出現問題也不知道要如何下手，因為一切都基於信任轉送公司
