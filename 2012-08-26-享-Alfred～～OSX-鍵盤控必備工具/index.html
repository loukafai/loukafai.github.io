▲ Alfred～～OSX 鍵盤控必備工具

假若你作為一名 [VIM](http://www.vim.org/) 愛好者，一定是相當喜愛使用鍵盤來操作電腦，剛好小弟也是一位重度愛用鍵盤之人，所以在這推一個 MAC OSX 上的殺手級軟體－[Alfred](http://www.alfredapp.com/)

● 取代 CMD＋Tab 切換程式

![Pastedgraphic-2](http://getfile5.posterous.com/getfile/files.posterous.com/loukafai/wHJ6KR4gVZ2kuOQaZaAAXg0zmiaLyo1v6e5eVwXCrXUXXU5395JHlQ3ONmzk/PastedGraphic-2.png)

其實一般用 [ cmd + tab ] 來切換使用中的程式已經很夠，但是當打開了的程式多了，往往要多按 tab 幾下才可以切到需要的程式，要是用了 [Alfred](http://www.alfredapp.com/) 之後，只需 [ Opt + 空格 ] 再配幾個字母就可以快速切換程式，甚至熟練之後可以達到「盲切」的境界，超級方便

* OSX 內建了 [Spotlight](http://support.apple.com/kb/HT2531) 也可以作程式切換，但是跟 [Alfred](http://www.alfredapp.com/) 相比速度慢很多

● 快速開／關 wifi

[![Pastedgraphic-3](http://getfile2.posterous.com/getfile/files.posterous.com/loukafai/YHAMS7CUxJOBLzz0Rl2Zo4HHyT1me4CFw6DQkSri48A7N7HwzlMt7Lhi4B0r/PastedGraphic-3.png.scaled.500.jpg)](http://getfile1.posterous.com/getfile/files.posterous.com/loukafai/TNGwP7qGIkQpSe9ytfyuqAL2tchIucuXQJp2biRNDq35jbtYkv4Ks8r78ewj/PastedGraphic-3.png)

安裝插件 Wi-Fi toggle 後就可以用關鍵字 [ wifi ] 來開關 MAC 的 wifi 功能

● 快速發 Tweet

[![Pastedgraphic-4](http://getfile8.posterous.com/getfile/files.posterous.com/loukafai/0EWPJFbUibSQc4fp23sJUh7xhDMrA0HZ18Bj04UyZgZhRomVHEqgchUDke40/PastedGraphic-4.png.scaled.500.jpg)](http://getfile7.posterous.com/getfile/files.posterous.com/loukafai/dhPiRaYx29Cm9ZI0FFWAmeSy7wTGCGKroVgIS1NeVFE8Otu5UqRUNMYfVcrQ/PastedGraphic-4.png)

安裝插件 AlfredTweet 就可以用關鍵字 [ tw ] 發 tweet
ps. 需要先作設定：(1) 安裝插件後輸入 ‘tw setup’ 到 twitter 取得授權 pin 碼 (2) ‘tw pin <your pin number>’

● 快速截圖

[![Pastedgraphic-5](http://getfile5.posterous.com/getfile/files.posterous.com/loukafai/i5v1i2FQQQGeOrVeXaU4mA7GNYKiR99haMNFfVZMn3NUKEQoKs2V8Acy6DgI/PastedGraphic-5.png.scaled.500.jpg)](http://getfile4.posterous.com/getfile/files.posterous.com/loukafai/2q5zNvJhWspNL6kexA7tVFhwIHN5gh4rGFuJBWGcGvmNLmq27xEAq1z2iYVl/PastedGraphic-5.png)

安裝插件 Screenshots with Alfred 後就可以用關鍵字 [ ss ] 快速截圖，雖然內建快捷鍵 [ cmd + shift + 4 ]，也很快但是需要用的手指會變多了，像我最常用的方式是 [ ss ic ] 把截圖複製到剪貼簿，快捷键是 [ cont + cmd + shift + 4 ] 就要用到 4 指
