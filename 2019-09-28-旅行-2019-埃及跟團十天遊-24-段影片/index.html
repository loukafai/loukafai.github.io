2019-egypt-10days, 2019-埃及, egypt, http://schemas.google.com/blogger/2008/kind#post, lift, trip, 埃及, 旅行, 生活, 非洲

之前在【2019 埃及跟團十天遊】的時候，有答應團友會拍在行程中錄下的影片跟大家分享，所以這篇就一次過把所有的影片集合在一篇裡面。

導遊說：「如果去過埃及最後只有手機只有一堆照片，沒有故事的話，埃及是白來了」，所以我就順便把導遊說的故事都拍下來了，給一路上拍到的 "石頭" 都有一段故事 XD

PS. 影片雖然是放在 Youtube 上，但是用了 "不公開" 設定，所以只有直接找到連結的人才能夠看到。

出發去埃及博物館

▲ 導遊在車上給我們做簡單介紹，經過解放廣場的，看看他是怎麼介紹的 ( via 20190401 出發去埃及博物館 - YouTube)

▲ 導遊的穆罕默德阿里清真寺介紹 ( via 20190401 阿里清真寺 - YouTube)

出發去金字塔

金字塔介紹

▲ 烈日當空之下，導遊仍然花時間給大家介紹金字塔的來源和作用，金字塔其實跟大家想像中的不一樣 ( via 20190402 金字塔介紹 - YouTube )

出發去坐埃及的火車

▲ 聽導遊介紹埃及的火車，然後埃及人的開車態度是多麼的隨性，開車撞到別人都不用擔心 ( via 20190402 出發去火車站 - YouTube)

夜臥火車

康孟波雙神殿

▲ 聽聽導遊講解，為什麼要幫鱷魚蓋一個神殿？原來神殿以前有治病的功能？ ( via 20190404 康孟波雙神殿 - YouTube)

艾得夫神殿

▲ 導遊很有趣準備了一些圖畫來講解，是經典的埃及神話故事 ( via 20190405 艾得夫神殿 - YouTube)

講解阿拉伯文化

▲ 回房間休息之前，導遊主動說要給我們上一堂，講解一下阿拉伯文化，在他們的世界裡，女人的身份地位真是比較低嗎？ ( via 20190405 講解阿拉伯文化 - YouTube)

伊斯納水匣門販賣

▲ 有拍了一段影片，是蠻有趣的體驗 ( via 20190405 伊斯納水匣門販賣 - YouTube)

路克索神廟

▲ 導遊在這裡給我們做了很精彩的講解，擁有 3400 年歷史的路克索神殿 ( via 20190405 路克索神廟 - YouTube)

遊輪表演肚皮舞和轉圈

▲ 晚餐之後游輪有安排了一段肚皮舞和轉圈給大家看。接著上場的 "轉圈" 大師好厲害，一直轉了幾分鐘都不會暈，我們都猜他下船以後就馬上在河邊吐 XD ( via 20190405 遊輪表演肚皮舞和轉圈 - YouTube)

帝王谷

▲ 同樣要聽講解，以前的法老在上任後只要做兩件事，1、蓋神殿，2、蓋陵墓。發現這個【帝王谷】的人花了二十年才找到，很累耶 ( via 20190406 帝王谷 - YouTube)

卡納克神廟

▲ 導遊講解關於【卡納克神殿】的歷史，為什麼可以蓋到那麼大？ ( via 20190406 卡納克神廟 - YouTube)

跟團去埃及

跟團的準備

第一天行程

出發 → 香港機場 → 約旦皇家航空 RJ 183 → 約旦安曼機場 → 約旦皇家航空 RJ 501 → 開羅機場

第二天行程

開羅機場 → 埃及博物館 → 午餐：Andrea Memphis Maadi → 穆罕默德阿里清真寺 → 薩拉丁城堡 → 晚餐：Citadel View Alain Le Notre → 飯店：Grand Nile Tower → 飯店兌換埃磅 → 飯店康樂設施

第三天行程

飯店早餐：Grand Nile Tower → 吉薩金字塔 → 金字塔最佳拍照點 → 獅身人面像 → 午餐：絲綢之路 → 哈利利市集 → 旅館盥洗 → 吉薩火車站 → 夜宿火車

第四天行程

亞斯文 → 亞斯文大水壩 → 登上尼羅河遊輪 → 游輪 Princess Sarah II → 游輪套房 → 游輪午餐 → 三角風帆船 → 亞斯文市集 → 游輪晚餐

第五天行程

起床咖啡 → 阿布辛貝神殿 → 納菲塔莉神殿 → 游輪午餐：Princess Sarah II → 游輪下午茶 → 康孟波雙神殿 → 鱷魚博物館 → 游輪晚餐：Princess Sarah II

第六天行程

坐馬車出發 → 艾得夫神殿 → 游輪早餐 → 伊斯納水匣門 → 小販叫賣 → 游輪午餐 → 游輪下午茶 → 路克索神殿 → 路克索市集 → 游輪晚餐 → 游輪肚皮舞表演

第七天行程

路克索熱氣球 → 游輪早餐 → 帝王谷 → KV6 → KV11 → KV2 → 曼儂巨像 → 卡納克神殿 → 購物點：沙草畫 → 午餐：埃及風味美食 → 出發去海紅 → 入住飯店 Mövenpick Resort Soma Bay → 房間 → 飯店晚餐 → 飯店舞蹈表演 → 飯店小商店

第八天行程

飯店早餐：Mövenpick Resort Soma Bay → 紅海玻璃底船和浮潛 → 玻璃底船景色 → 紅海浮潛 → 回飯店放鬆 → 飯店午餐：Mövenpick Resort Soma Bay → 四輪傳動車貝都因村落 → 爬沙漠 → 沙漠下午茶 → 飯店晚餐 → 飯店表演節目

第九天行程

飯店退房 → 午餐：Queen Cleopatra Restaurant & Coffee → 購物：香精店 Golden Eagle Crystal → 開羅機場 → 約旦皇家航空 RJ 506 → 約旦安曼機場 → 約旦皇家航空 RJ 182 → 曼谷中停加油 → 港珠澳大橋
