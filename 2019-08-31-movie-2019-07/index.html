摩天營救 Skyscraper (2018)

via 摩天營救 (豆瓣)

放了好久的電影，每次都是看到一半就卡掉，也許是緣份未到，不然就是主角不夠吸引。

雖然主角是 The Rock，不過這次他演的是受傷了的退役軍人，缺了一隻腿。不過還是一樣的厲害，上山下（火）海都難不倒他，所以到最後還是一樣大團圓結局，全身而退。《摩》的特效做得還算可以，只是劇情真的很普通，如果可以再多點驚喜或笑點會更好。

這部只適合 The Rock 的粉絲和動作電影迷。

進擊的巨人 第三季 Part.2 進撃の巨人 Season3 Part.2 (2019)

via 進擊的巨人 第三季 Part.2 (豆瓣)

追了好久的動漫，亦是目前唯一有在看的日本漫畫，原因是劇情有夠熱血，有些部分的劇情對人生很有啟發，我想這樣是【進擊的巨人】能夠一直紅到現在的主要原因。

這季有點特別是分為了上下兩 Part，沒有直接分開做兩季，可能是劇情有比較連貫性，分開兩季會有點奇怪。

我很喜歡這季的《進》，除了因為戰鬥劇情很緊湊以外，還多了一些感人的元素，裡面有講前線的調查兵團為了引開野獸巨人的注意而犧牲自己、團長放棄了重生的機會把藥針讓給阿尼。我會覺得如果角色是以這樣的方式離去，雖然有一種可惜，但是會有振奮人心的作用，我們在現實生活中不是也充滿很多「犧牲」嗎？我們犧牲時間勞力去工作換取金錢，父母犧牲時間心血去照顧小孩。這樣的犧牲並不是毫無意義，是為了讓後人活得更好。

推薦這部給喜歡看動作和喜歡深入一點思考的朋友。

P風暴 (2019)

via 反貪風暴4 (豆瓣)

又是一部以香港 ICAC 為主角的反貪電影，已經是同系列的第四部，古天樂很厲害地連任了調查主任，這次他更跑進了監獄當臥底，電影完全迎合了香港電影迷最喜歡的元素：臥底＋警匪。

想不到古天樂在拍完《殺破狼·貪狼》(2017)之後，把自己進化成動作演員，武打戲份越來越重，超會打，粉絲看到不知道會是高興還是心痛？

講回電影，覺得最好看的是張繼聰，上一集他是演小混混慘死收場，這次終於有好一點，雖然同樣是個小混混但是仍然有點正義感，幫助古天樂破案。記得劇情中有一段他說香港年青人的出路就是坐監，因為不用房租又包吃住，所以他要每隔一段時間就要回來坐監，充滿黑色幽默。

這部適合給喜歡看充滿正義感電影的朋友。

我們來做體操吧 體操しようよ (2018)

via 我們來做體操吧 (豆瓣)

看到某視頻網站一天上架了這部，看標題覺得應該蠻有趣，加上草刈正雄和木村文乃都是喜歡的演員，所以拿下來看看。

電影風格又是那種典型的治癒系電影，講一個退休老頭無所事事，偶爾加入了一個晨運社團，結果越玩越認真，認真過頭就跟團友鬧翻，這時候才發現自己似乎過活了一半輩子，其實不太會跟別人相處。

作為日本電影，最後應該是要有快樂結局的，所以最終男主跟團友和好，跟女兒的關係也得到好轉。覺得電影就是要表達做人不要太執著，懂得開通，這樣你的晚年就會過得比較輕鬆。

這部推薦給喜歡日本溫情電影風格的朋友。

一個明星的誕生 A Star Is Born (2018)

via 一個明星的誕生 (豆瓣)

去年很紅的音樂電影，主要原因應該是 Lady Gaga 出演女主角吧！我想就算沒有看過電影，也應該有聽過主題曲《Shallow》，這首歌早就被播到街知巷聞了。

原以為《 一個明星的誕生》會像之前的音樂電影《樂來越愛你》那樣，即使沒有大團圓結局但是也不要希望太悲劇。《一》的前半部份太直白套路就算，最後男主還給我自殺，真是負能量十足。從電影中只能體會到，珍惜身邊愛人，因為他（她）可能隨時會不見，永遠留下的是最美妙的歌聲。

所以，這部只推薦給喜歡 Lady Gaga 和音樂電影的朋友。

輕松小熊和小薰 リラックマとカオルさん (2019)

via 輕松小熊和小薰 (豆瓣)

去旅行時看的劇，是 Netflix 的原創創，全季只有短短 11 集，每集更只有十多分鐘，非常好上手。整部劇沒有什麼劇情可言，就是有夠療癒。看著輕松小熊和主角小薰的互動，慵懶又好笑，絕對是能夠給上班族的觀眾得到一份心靈慰藉。

這部絕對要推薦給喜歡輕松小熊和卡通的朋友。

阿麗塔：戰鬥天使 Alita: Battle Angel (2019)

via 阿麗塔：戰鬥天使 (豆瓣)

會知道這部電影，是因為手機小米 9 在發佈時同時有推出電影同款的「戰鬥天使版」，十分厲害。手機雖然沒有買，但是電影還是要看，尤其科技電影完全是我的菜。

老實說，《阿》劇情很一般，但是電影的特效做得非常出色，女主阿麗塔的整個臉都是 CG，身體的機械手腳更不用說了，而且裡面每個機械角色的特效都做得唯妙唯肖，場景也幾乎是「做」出來的吧！但是電影畫面整個很流暢，毫無違和感，光是看特效就很值得了。

這部絕對要推薦給喜歡科幻動作電影的朋友。
