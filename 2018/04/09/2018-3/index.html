2018 年 3 月影視清單，個人觀影後的排名如下：

  1. Unnatural 法醫女王 (2018)
  2. 追龍 Chasing the Dragon (2017)
  3. 與神同行 Along With God (2017)
  4. 完美巨聲幫 3 Pitch Perfect 3 (2017)
  5. 加勒比海盜：惡靈啟航 Pirates of the Caribbean: Dead Men Tell No Tales (2017)

與神同行 Along With God (2017)

via 與神同行 (豆瓣)

人氣韓國電影，由漫畫改編而成，故事是說人在死後要接受審判，成功的話就可以去投胎，失敗的話當然要繼續受苦囉！很合理。

電影是用了死神救孝子的故事來包裝，感人的故事背後的目的是要導人向善，就如同故事所講，如果一個人生前做得越多壞事，去到地獄之後會受到更多的苦難來償還，所以觀眾們都要注意自己平常的行為舉止。

當然，你信不信有地獄、輪迴這些東西又是另外一回事了。

個人覺得這部【與神同行】的劇情尚算不錯，但是電影特效做得很出色，如果不喜歡這種催淚的電影，看看韓國人製作的特效也蠻棒的。

潘多拉：永恆的生命 パンドラ～永遠の命～ (2014)

via 潘多拉：永恆的生命 (豆瓣)

清庫存所看的戲，故事是講主角（堺雅人 飾）把自己基因複製，生出一個跟自己基因一樣「兒子」，然後帶著兒子逃亡。

故事聽起來真是還好，所以重點當然是落在主角堺雅人的演出，電影是 2014 年的作品，就是他演完【半泽直树】（2013）之後拍的，明顯是想要憑他的人氣幫電影票房衝一把，然後再拍個續集大賺一筆，結果當然沒有囉！因為劇情真的太普通了，當然也有可能是堺雅人走紅之後檔期滿滿。

要說劇情一個有趣的點，就是說兒子在小小年紀竟然會殺人，然後說這是他基因裡一直存在的機制，叫自私基因，看起來真的有點恐怖！差點以為是郭富城之前的那部【殺人犯】的再版，哈哈！

加勒比海盜：惡靈啟航 Pirates of the Caribbean: Dead Men Tell No Tales (2017)

via 加勒比海盜5：死無對證 (豆瓣)

不知不覺，原來這部經典的海盜電影已經拍到第五部續集，海盜世界的故事真是十分豐富，長拍長有的樣子，只是觀眾們好像越看越膩，這點在電影的評分可以看得出來。

講回這部第五集，這集的主要角色換了一半，只剩下杰克船長在撐，看到杰克船長的老拍檔威爾的兒子都長到那麼大了，杰克船長還是那麼年青，實在太神奇。

這集好不好看？如果是這個系列的老粉絲一定要看啦！只是單純以電影來說，除了特效和畫面仍然保持荷李活一貫的水準之外，還真是有點普通。

Unnatural 法醫女王 (2018)

via Unnatural (豆瓣)

個人本季最推薦的日劇，網上的好評很多，是一部良心劇，劇情是利用了受害人的角度去反映社會上的存在的問題，應該帶給觀眾蠻多反思的。

當然，本人最支持的石原里美也是看點之一啦 XD

在十集之中，個人對蜜蜂蛋糕工廠的那集較為深刻，因為自己也是打工仔嘛 XD 那一集的劇情是反映社會上普遍過勞工作的問題，但是老闆不會在乎、消費者也不會理會，僱員就只能想辦法自保，職業安全記得要遵守！尤其是當你身後還背負著養活一個家庭的責任。

值得一提的還有日劇裡面的插曲：米津玄師 -「Lemon」 真的好好聽，相信為日劇的劇情和氣氛帶動加分不小。

追龍 Chasing the Dragon (2017)

via 追龍 (豆瓣)

假日用來娛樂一下的電影，發現意外的精彩好好看。

經過那麼多年，發現甄子丹的演技改進了好多，他演的跛豪有情有義、勇猛、執著，電影把他拍得十分立體，有血有肉的感覺；而劉德華就一如既往的入型入格，他演的華探長在戲中精明能幹，只是電影沒有在解釋他為什麼那麼厲害，可能是因為「五億雷洛」的故事已經相當經典，重點放在跛豪身上。

跛豪的一生，「義氣」二字成為了他成敗的關鍵，他為了救雷洛失去了一隻腳，又為了幫弟弟復仇害死了幾個陪他出生入死的好兄弟。

有時候會討厭那些個性很執著的人，但是偏偏就是有一些無謂的堅持，成就了他們傳奇、無怨無悔的人生。

完美巨聲幫 3 Pitch Perfect 3 (2017)

via 完美音調3 (豆瓣)

不知不覺，【完美巨聲幫】來到了第三集，既然能夠拍到三集，想必是因為前兩集的成功。

沒錯！【完美巨聲幫】的第一集真是很精彩，雖然電影的劇情平平，但是勝在歌曲好聽，讓大家都認識到「無伴奏」合唱；至於第二集，也只是延續了第一集的人氣，沒有太多突破；而這次的第三集，就多一些動作場面，理應是多了一種新鮮感，但是問題就在改變音樂的劇情佔比就會相對弱化了歌唱部分，變得頭重腳輕。

喜歡聽歌的朋友，這集依然值得一看，歌曲同樣的好聽，如果不介意電影劇情有點無聊的話啦！
