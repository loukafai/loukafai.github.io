2018 年 1 月影視清單，個人觀影後的排名如下：

  1. 雷神3：諸神黃昏 Thor: Ragnarok (2017)
  2. 邊緣人 教你制作情婦的方法 (2017)
  3. 電鋸驚魂8：豎鋸 Jigsaw (2017)

邊緣人 教你制作情婦的方法 (2017)

via 邊緣人 フリンジマン～愛人の作り方教えます～ (2017)

這是部深夜時段的日劇，劇情是和標題所寫的一樣，是教人怎樣去找情婦，聽起來很吸引是吧！

一開始是以為這種劇名的日劇一定是標題黨，但是看了幾集之後就知道劇情真的是找情婦，日劇裡面有講一些所謂分析哪幾種女性的特徵是有機會成為情婦，那些分析的角度都蠻有趣，像什麼直接稱呼你名字的同事會比較有機會、提示你領帶打歪了的女性也有可能，反正這種荒謬搞笑取向的日劇用來笑一笑就好。

電鋸驚魂8：豎鋸 Jigsaw (2017)

via 電鋸驚魂8：豎鋸 Jigsaw (2017)

相信很多喜歡重口味電影的朋友跟我一樣，對《Saw》系列一直都讚口不絕，2004 年的《Saw》及 2005 年《Saw II》都非常經典，電影以低成本和少量場景去拍攝，但卻憑著出色的劇本拍出令人恐懼的效果，尤其是電影那種設計巧妙的情節，每集的劇情都每每都經過一輪的扭轉，到最後才令人恍然大悟，但偏偏又令人對故事有一種信服，真是十分出色。

《Saw》系列來到第八集，雖然網上觀眾對這一集的評價不高，但個人還是覺得可以收貨的啦，因為在這集裡面喜歡和想看的《Saw》元素都包含了，第八集裡面有精心設計的殺人關卡、最後的情節有個轉折位，當然，跟前七集比較起來還是有點遜色啦！因為經過七集，可以拍可以想的都己經拍出來了，只不過個人覺得續集能夠把《Saw》應該要有的東西拍出來就已經很棒，足夠用來回味了。

雷神3：諸神黃昏 Thor: Ragnarok (2017)

via 雷神3：諸神黃昏 Thor: Ragnarok (2017)

老實說個人對前兩集的雷神的評價都很一般，但是這部第三集卻給了我一種耳目一新的感覺。一來是多了浩克和神奇博士的加入，二來是劇情編排上多了一種幽默感，整部電影看起來比較有趣味。

還有另一個喜歡的點是電影的背影終於離開了地球回到了雷神的家鄉 Asgard，還有去了一個神奇的 Sakaar 星球，就是有了場景的改變才顯得電影比較精彩，所以這部《雷神3》真是可以推薦一看。
