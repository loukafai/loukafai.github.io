2018 年 4 月影視清單，個人觀影後的排名如下：

  1. 行屍走肉 第八季 The Walking Dead Season 8 (2017)
  2. 逃出絕命鎮 Get Out (2017)
  3. 通勤營救 The Commuter (2018)
  4. 殺破狼‧貪狼 Paradox (2017)

殺破狼‧貪狼 Paradox (2017)

via 殺破狼·貪狼 (豆瓣)

2017 年的電影等到現在才看，當然是為了古天樂的【香港電影金像獎 - 最佳男主角】。

看完覺得古天樂的努力真是值得嘉獎，因為他由一個不太會打的演員去演一個不停打架的角色，絕對是一個挑戰，如果說動作戲是個加分項的話，這點大大加分。

不過覺得同期《追龍》裡的甄子丹也演得不錯呀！他跟古天樂二選一的話我可能會選甄子丹。主要因為同樣是性格執著的角色，甄子丹演的跛豪比較有層次，飽經風傷的男人總是比較受人欣賞。

最後是《殺破狼·貪狼》的打斗戲，這集的動作場面是比較多，但是沒有第一集的好看和經典，也沒有第二集的張晉打得那麼凌厲，是比較可惜的一點，不過愛看動作戲的朋友也不會失望，因為有 Tony Jaa 就值回票價啦！

通勤營救 The Commuter (2018)

via 通勤營救 (豆瓣)

被稱為「神奇老爸」 - 《Taken》系列的姐妹篇，當然沒有《Taken》第一集那麼驚喜，但是場面和製作方面也絕對沒有偷工減料。

這集的老爸比較可憐，他這次不是救人而是救自己，因為他被工作了十年的公司解僱，他需要錢去生活，這時候一個賺大錢的機會來到，他會不會為了錢而做出違背良心的事？

這部《通勤營救》好看是在困獸鬥的背景，大部分的情節是在電車上發生，動作場面不多但劇情相對緊湊，只是背後的陰謀有點牽強，犯罪集團看似強大得無所不能，但是最後又被男主角打敗。不過作為爽片的話，看開心就好，不能計較太多。

逃出絕命鎮 Get Out (2017)

via 逃出絕命鎮 (豆瓣)

衝着網上的好評來看這部電影，覺得是部不錯的爆米花電影，具娛樂性，但是好像距離十分精彩又似乎差了一點。

電影是看起來有點類似解謎的感覺，觀眾跟著主角一步一步探險，直到發現正妹女友原來是來陷害自己的時候，就跟著主角一起逃出生天，這種感覺真的蠻爽。

覺得如果劇情安排男主角逃出險景的過程可以再長一點，再困難一點的話，也許會更好看。

行屍走肉 第八季 The Walking Dead Season 8 (2017)

via 行屍走肉 第八季 (豆瓣)

作為鐵粉，給這部是本月第一名完全不過分，個人認為這季算是精彩，沒有讓粉絲失望，尤其是結局最後的高潮部分，看到 Rick 終於獲勝了就足夠開心。

Carl 的離去是有點可惜，但是這天始終都會來到，只是來得有點早，他的離去也確實推進了這季的劇情，他把遺言寫進信裡的情節是相當感人，有點《P.S. 我愛你》的感覺。

這部來到第八季，收視率下降，可能很多人都已經棄劇，不過個人還是會堅持收看，因為覺得《行屍》在某程度是反映社會的小縮影，人與人之間就是會因為很多意見不和矛盾產生爭執，看這部美劇偶然會帶來一些啟發。
