[旅行] 2019 埃及跟團十天遊 11～Day7-1：路克索熱氣球，帝王谷

2019-09-11
2019-egypt-10days, 2019-埃及, egypt, http://schemas.google.com/blogger/2008/kind#post, lift, trip, 埃及, 旅行, 生活, 非洲

【2019 埃及跟團十天遊】來到第七天。坐了三天的游輪，今天終於依依不捨地離開。但是不用灰心，今天有要參觀重頭戲【帝王谷】。

【帝王谷】都是千年古墓，裡面有漂亮的壁畫，不像金字塔裡面黑漆漆的什麼都沒有。而且現在的【帝王谷】可以付費拍照，以前是只能買相片的呢！喜歡拍照的朋友可以盡情地拍個過癮。

第七天行程

路克索熱氣球 → 游輪早餐 → 帝王谷 → KV6 → KV11 → KV2 → 曼儂巨像 → 卡納克神殿 → 購物點：沙草畫 → 午餐：埃及風味美食 → 出發去海紅 → 入住飯店 Mövenpick Resort Soma Bay → 房間 → 飯店晚餐 → 飯店舞蹈表演 → 飯店小商店

路克索熱氣球

▲ 因為一些原因，香港的旅行團都沒有【路克索熱氣球】，所以參加台灣的旅行團都有機會坐一下這個熱氣球，只是費用要 200 美金每人，而且聽一些團友說沒有土耳其的熱氣球厲害，所以我們就沒有參加。但是仍然早早起床看看別人坐的熱氣球。

▲ 看到遠方的氣球不過有點遠，手機的話應該要等到華為 p50 （如果有的話）才能拍到那麼遠

游輪早餐

▲ 6:42 就去吃早餐，好像平日上班都不會那麼早起 XD

▲ 時間是 7:54，大家吃完早餐後出發去看今天的重頭戲【帝王谷】

▲ 坐小船到對岸之後，轉塔巴士

▲ 真邊真的有夠荒蕪，半棵樹都沒有，誰會想到這裡收藏了那麼多寶物

帝王谷

帝王谷，是位於埃及埋葬古埃及新王國時期18到20王朝的法老和貴族的一個山谷，它坐落在尼羅河西岸的金字塔形山峰庫爾恩之巔，與底比斯相對（現在的盧克索）。帝王谷分為東谷和西谷，大多數重要的陵墓位於東谷。西谷只有一座陵墓向公眾開放：伊特努特·阿伊的陵墓（圖坦卡蒙的繼任者）。西谷也有多處其他重要的墓葬，包括阿蒙霍特普三世，但是仍然在發掘過程中並尚未向公眾開放。 ( via 帝王谷 - 維基百科，自由的百科全書)

▲ 就是說新王國時期的法老都不蓋金字塔，大家死後一起「享用」這個長得像金字塔的山谷，又不會容易被人發現

▲ 這邊的墓那麼多，可以買個 pass 看個夠 XD

▲ 【帝王谷】模型，看到埃及在山裡面掘了很多洞，而且掘得很深入呢

▲ 還要再坐小車進入園區

▲ 想在【帝王谷】內拍照同樣要加買拍照券 300 EGP，難得來到當然不能省這種錢。而且，我們跟團來的根本沒有時間好好欣賞，拍回家慢慢回味也是不錯的……只是，會想要回味也是好幾年之後吧 XD

▲ 同樣要聽講解，以前的法老在上任後只要做兩件事，1、蓋神殿，2、蓋陵墓。發現這個【帝王谷】的人花了二十年才找到，很累耶 ( via 20190406 帝王谷 - YouTube)
