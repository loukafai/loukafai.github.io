綠簿旅友 Green Book (2018)

via 綠皮書 (豆瓣)

既然是 2019 奧斯卡金像獎最佳影片，當然不能錯過！

對比起 2018 的奧斯卡最佳電影《水底情深》（英語：The Shape of Water，中國大陸、馬新譯《水形物語》，香港譯《忘形水》），覺得這部《綠簿旅友》是沒有那麼「重口味」，是一般大眾比較能接受的類型。

所以我個人的話會比較喜歡今年的《綠簿旅友》，就是因為電影佈局看起來比較平宜近人，就講兩個角色背景非常有矛盾的兩個人在旅行途中慢慢建立信任，一種特別的友誼，同時電影有種公路電影的 feel，如果你個人又是喜歡去旅行的話，會看得很開心。

毒梟 第三季 Narcos Season 3 (2017)

via 毒梟 第三季 (豆瓣)

這季只剩下主角潘那一人來對抗「卡利集團」，幾乎是一個不可能的任務，但是看著主角一步一步地把這個組織規模相當龐大的販毒集團擊潰，整個熱血感滿滿，難怪《毒梟》所有三季在美劇圈得到的評價都那麼高。

第三季在故事方面多了一個新角度，其中一個故事分支是由反派大本營的裡面的安全顧問 Jorge Salcedo 去切入，剛剛開始會覺得奇怪，原來他是本季重要的角色，因為演到最後幾集會他會背叛整個組織，為了家人的生活安全，他不得不出賣老大。

當一個組織裡面出現背叛者，基本上就代表他們差不多要完了。那麼 Jorge Salcedo 為什麼要背叛？是因為他本性善良？良心發現嗎？都不是，他只不過是看到首領大勢而去，做出他認為最合適的選擇。大家的人生不都是這樣嗎？

強烈推薦《毒梟》給喜歡美劇的朋友。

唐山大地震 (2010)

via 唐山大地震 (豆瓣)

收藏了好久的電影一直不敢打開，趁著一天早上起床，天色陰陰的，情緒有點上來，就終於鼓起勇氣打開來看看，電影真是感人，看得我眼睛都紅起來了。

在劇情中，當時在瓦礫之下，母親李元妮在兒子和女兒之間選擇了救兒子，這樣是重男輕女，非常符合中國人的傳統，因為社會環境普遍都是這樣，有什麼問題？

只是如果你是那個被放棄的女兒，會怎樣想？生氣是很正常不過的表現，所以即使在她長大懂事之後，都沒有想要回去找自己的母親。

當然，人的心態是會改變，可能是她自己當了母親以後，才了解到母親的難處，也因為在機緣巧合的情況下與親人重逢，最後才選擇原諒自己的媽媽，不然我想女主還是會恨她媽媽一輩子。

這部電影比較沉重，不是像一般的災難電影都在宣揚"愛要及時"之類的大道理，這部是切切實實的一部記錄傷痛的電影，請小心服用。

葉問外傳：張天志 (2018)

via 葉問外傳：張天志 (豆瓣)

張晉算是喜歡的動作演員，由他在《一代宗師》和《殺破狼 2》裡面的演出就覺得這位演員頗有型格，功夫打起來也好看，就是演技和表情上比較木納。

這次張晉由配角升級為主角，而電影公司更加是重本投入，找來楊紫瓊和 Tony Jaa 一起演，份量十足。

整體來說這部《葉問外傳：張天志》相當收貨，該打的都有打，劇情發展順暢無明顯破綻。唯一不足的點是男主只在最後一場才把詠春使出來，雖然這樣的安排是有比較帶出高潮一幕，但是觀眾裡應該有超多人是想看詠春的吧！

還有 Tony Jaa 的打戲好少，最後他應該要和張晉一起打大反派才對呀！

推薦這部給喜歡香港武打動作電影的朋友，希望之後會有第二集囉！

世界是平的 Outsourced (2006)

via 世界是平的 (豆瓣)

又是一部收藏好久的電影，當初是看一篇講：如何在工作中找回方向的文章初介紹，才認識這部。

Outsourced 外判曾經是十幾年前的熱話，【外判】是潮流趨勢沒有人能夠避免，只是像男主那樣一開始由外派去印度的經理，到最後喜歡上當地的女職員，在劇情上的安排是非常合理不過，好像是說除了工作機會以外，甚至連挑情人都可以外判，如果在自己的國家找不到女朋友，說不定出去別的地方走一走，未來伴侶就隨時在身邊出現。

電影有趣的一個點是可以看到一些印度文化，看完電影都想要去印度旅行一下。

賣房子的女人的逆襲 家売るオンナの逆襲 (2019)

via 賣房子的女人的逆襲 (豆瓣)

這個之前有看第一季，是一部很有特色日劇，看著漂亮的北川景子飾演一個臉無表情的地產銷售員，不管顧客有多麼特別的要求，也不論多麼奇怪的房子，她總是可以順利把房子賣出，因為「我是房屋銷售員！」是她一直掛在嘴邊的口頭禪。

如果劇情只是一直演賣房子的戲碼一定會很無聊，所以來到第二季，加入了松田翔太這個新角色，他是北川的前情人！？又是北川的對手。

更好笑的是松田翔太又與千葉雄大展開了一段 Gay 情，充滿笑料。

《房仲女王》是最近看過蠻有特色的日劇，劇情輕鬆有趣笑料多，日劇迷不要錯過。

大黃蜂 Bumblebee (2018)

via 大黃蜂 (豆瓣)

電影公司沒有再拍《變形金剛》的續集是有點失望，不過《變》之前幾集的口碑和評價都是大家有目共睹的……. 所以這次的《大黃蜂》希望能夠帶來一點新意。

這次的主角改為女生，劇情上改以感情為主要切入面，那些精彩的爆破特技場面改為次要，的確是可以帶來一點新意，不過就會讓想要看動作戲的朋友（例如我）有點失望，整部戲就只有初段大約 20 分鐘吸引眼球，有很多機械人登場，又有大戰場面；不過到中段就以文戲為主，交待女主和大黃蜂的感情發展；結尾有一段打架的戲，尚算精彩。

最後，覺得電影公司希望透過《大黃蜂》為整個系列來個重啟，擺脫這個系列過往以來一味無腦打鬥爽片的印像，老實講，像《變形金剛》二到五集那種一直充滿快速機械人快速變身和打鬥的戲，頭半小時會看得很爽，但是繼續看就會很無聊。說到底就是因為這些都是欠缺了情感的打，沒有靈魂。

要打得好看，就要故事背景有感情，有故事的打，而到底人類與外星機械人的感情能夠建立多深，打得多有感？就要看編劇的功力了。

炸雞特攻隊 Extreme Job (2019)

via 極限職業 (豆瓣)

看網友介紹是一部有趣的電影，看完就會想吃炸雞，評分也夠高，所以便找來看看。

如果一個人當警察是為了賺錢討生活的時候，突然得到有比繼續做警察賺更多錢的機會，那麼，放棄當警察是很自然的事吧！

還好，社會上是有很多充滿正義感的朋友，全靠他們的保護，社會才能得到安穩。

只是我想，為什麼炸雞生意不能和當警察並存呢？警察也是人呀，他們也要生活，既然可以做出受歡迎的食物，要用來賺錢當然是十分合理的。

最後個人覺得這部《炸雞特攻隊》好看程度只是一般，笑點是有，動作戲也有，電影會紅起來可能是有炸雞店家想要順便做廣告。
