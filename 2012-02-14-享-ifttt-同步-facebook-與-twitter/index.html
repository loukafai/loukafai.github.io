▲ 分享一個使用 ifttt 同步 facebook 與 twitter 的方法
> 最近正式開始用 twitter，想要用 twitter 的原因是它在於「純文字訊息」傳播和交流的方式比 facebook 較強，而且高手們都愛用 twitter。

雖然不喜歡 facebook 但是可惜偏偏不能完全放棄它，所以辦法就是讓 facebook 與 twitter 同步。其實類似的方法是用 ping.fm ，但是 ping.fm 的使用方法需要登入它們的網頁，而我想要的是「把我在 facebook 上寫的東西直接同步到 twitter」，不止是用電腦發的 facebook，連手機也上發的也可以。

使用方法：

ifttt的工作原理是：「如果這樣….就做那樣」，所以我們可以利用這個特性去做 facebook 對 twitter 的轉發；告訴 ifttt 如果你在 facebook 寫了一個新的近況或發了一張新的照片就同步到 twitter
