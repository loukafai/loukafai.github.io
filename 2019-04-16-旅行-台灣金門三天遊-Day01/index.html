[旅行] 台灣金門三天遊~~Day01

2019-04-16
2018-kinmen-3days, 2018-金門, life, lift, taiwan, trip, 台灣, 旅行, 生活, 金門

2018 年 11 月份去了台灣金門來個三天兩夜小旅行。

旅程心得是，覺得金門是個很充滿歷史和回憶的地方，很適合長者來這裡懷緬過去、也適合年青人來學習戰爭的殘忍，認識世界和平真的很重要。

再說，金門也有很多美食例如蚵仔麵線、蚵仔煎、牛肉麵都跟台灣本島吃到的風格不同，尤其是蚵嗲更是值得來嚐嚐。

第一天行程

松山機場 → 金門 → 金城車站 → 水頭碼頭 → 烈嶼（小金門）九宮碼頭 → 九宮港區旅客服務中心租電動車 → 午餐：黃厝三層樓民宿 → 勇士堡/鐵漢堡 + 地雷主題館 → 湖井頭戰史館 → 八達樓子 → 后麟步槍模擬射擊館 → 沙溪堡 → 九宮(四維)坑道 → 坐渡輪回金門 → 金豐租車 → 住宿：金門縣三元民宿 (San Yuan Guest House) → 小吃：蚵嗲之家 → 小吃：金門蛋捲 → 晚餐：米香屋

機票

立榮航空

去程： 0740-0920 (B7-8805) $1598

回程： 1730-1840 (B7-8826) $1522

松山機場

▲ 因為是早上七點多的飛機，當日早上四、五點就起床由桃園出發。個人是第一次經由松山機場出發

▲ 沒有想到坐個國內飛機也要出示入台證，有夠麻煩的

▲ 出發沒有空橋，要坐接駁車

▲ 飛機 ATR-72 是使用兩個雙螺旋槳的民航機，個人是第一次坐，感覺舒適度跟使用雙引擎的飛機差別不大

▲ 兩人坐位

▲ 遠東的 MD-82/83 飛機也有興趣試試

▲ 沒有想到空姐的圍裙也有賣，用來 cosplay 應該很有情趣 >///<

▲ 看到 101

▲ 因為好快就到金門，所以沒有餐點，只有一杯咖啡

▲ 0730 起飛，0900 就到金門，飛了大概 1 個半小時

金門

▲ 馬上坐公車去【金城車站】

金城車站

▲ 金城車站有個旅客服務中心可以免費寄放行李，十分貼心！！！

▲ 再由【金城車站】坐公車去【水頭】

水頭碼頭

▲ 刷悠悠卡 60 元就直接坐上渡輪，機車可也可以一起去

參考：金門縣公共車船管理處-班船時刻-班船時刻

▲ 金門本島連接烈嶼（小金門）的跨海大橋 - 「金門大橋」，由 2012 年開始到今年 2018 年還在建，預計是 2020 年 9 月完工，加油！！

烈嶼（小金門）九宮碼頭

▲ 有太陽能燈

▲ 最近有慶典

九宮港區旅客服務中心租電動車

▲ e-moving 電動機車 6 小時租金是 150 元，去指定的店家消費之後更可以折抵 50 元，十分划算

午餐：黃厝三層樓民宿

▲ 我們不是來住宿，是要吃早午餐。【黃厝三層樓民宿】是熱門的餐廳，早點來吃就不用排隊了

參考：黃厝三層樓民宿 - Google 地圖

▲ 好多金門高樑酒

▲ 金門的地雷被排光了，這些警告旗子就成為了回憶

▲ 蚵仔煎 100 元，滿滿的蚵仔，非常鮮味

▲ 簡餐一份，有干貝蚵乾飯，紫菜湯，芋泥丸子，150 元，這樣小小的一份剛剛好，想試的東西都有

▲ 芋泥招牌冰 70 元，正妹就是說對這家的芋泥冰念念不忘，說超級好吃。我覺得也非常香甜，值得推薦

▲ 喜歡還有伴手禮可以買，都是金門限定的呢

勇士堡/鐵漢堡 + 地雷主題館

▲ 像真度太高，覺得每個經過的人都會被阿兵哥嚇到

烈嶼三堡之一的勇士堡，占地面積廣闊，進出碉堡需要經過一座小橋，穿越三公尺深的壕溝與緩長的斜坡，在偽裝網之後就是寫著勇士堡字樣的大門，重重關卡足顯當時烈嶼處於最前線的緊繃氛圍。參考：勇士堡 | 金門觀光旅遊網

▲ 這張是要表達 "勇士" 的概念

▲ 先逛地雷主題館，是講以前金門有很多地雷，花了很多時間才清完，同時世界上很多國家目前都仍然有地雷，好多無辜的平民百姓被炸到

▲ 到處可見的提示標語

▲ 最後經由【鐵漢堡】出來

▲ 金門人的房子都好大好漂亮，是土地多的好處耶

湖井頭戰史館

湖井頭戰史館，藉此讓駐守過金門的退役軍人，回到本館時能憶起其當兵時的歷程；讓沒當過兵的人了解在金門當兵的辛苦。參考：湖井頭戰史館 | 金門國家公園管理處全球資訊網

▲ 用個望遠鏡就可以看到對岸的廈門，連岸邊有幾個人在騎腳踏車都看得很清楚

▲ 「還我河山」，這樣大大的標語在台灣本島沒有看到，也算是金門的一個特色

▲ 「自力更生」

八達樓子

在烈嶼九井路與八青路交會的圓環，有一座仿城堡式的建築，樓頂上塑有7位分據四方，手握手榴彈或機關槍的英勇戰士雕像，就是著名景點八達樓子，這是為了紀念1933年抗日戰爭中，駐守在長城古北口八達樓子的7位勇士而建。參考：八達樓子 | 金門觀光旅遊網

勝利門是一座白色圓拱型建築物，地點位於小金門湖下村的路口，之前是砲彈造型的八二三抗戰紀念碑，為了紀念當年在八二三砲戰國軍勇敢抗戰26萬餘發砲彈的凶猛攻擊後所取得的光榮勝戰而打造。參考：勝利門 | 金門觀光旅遊網

后麟步槍模擬射擊館

穿上迷彩服、扛起步槍，來到烈嶼別忘記來場步槍模擬射擊體驗！后麟營區步槍模擬射擊館位於烈嶼島上東北角的后頭，串接烈嶼主要道路，交通十分便利。

▲ 有先打電話預約了步槍模擬射擊，原本全票玩一次要新台幣350元，但是現階段開放免費體驗！！ 一次大概玩 20 分鐘，蠻有趣的

▲ 開始之前教官講解一下就可以開始。雖然只是模擬射擊，但是和槍是一樣的重量和質感，也會像真槍一樣卡彈 XD

▲ 更可以借軍服和假槍 cosplay 一下

▲ 玩之前要在綠幕前拍照記錄，然後會在電腦幫你 key 到金門的各個景點

▲ 成績單，要是覺得不夠過癮可以在現場候補再玩一次

▲ 如果喜歡上場殺敵的感覺，更可以現場報名去當兵呢！！所以這是國防局贊助的活動？

參考

  1. 后麟模擬靶場 - Google 地圖
  2. 后麟步槍模擬射擊館 | 金門觀光旅遊網
  3. 金門旅遊-烈嶼后麟步槍模擬射擊館 @ 留在金門忘了飛 Visit Kinmen :: 痞客邦 ::

沙溪堡

沙溪堡位於青岐村外的岬角上，主要是由花崗片麻岩及花崗岩混合而成，從南山頭角度觀看，像極了鳥嘴，所以又稱為「鳳嘴」。 參考：沙溪堡 | 金門觀光旅遊網

▲ 坑道就是當年軍人用來抗敵

▲ 想像以前軍人要一直在這裡防備守護國家，很偉大

▲ 廈門就在對面

▲ 【沙溪堡】風景很好

▲ 11 月份來到金門旅遊，天氣微涼剛剛好

九宮(四維)坑道

大金門有翟山坑道，小金門則以九宮坑道當家，同樣是戰時為確保軍事運補作業的安全，採取的地下化作戰設施，1965年完工的九宮坑道，從九宮碼頭為起點穿透山體，一路延伸到羅厝漁港。 參考：九宮坑道 | 金門觀光旅遊網

寬廣、機關縝密的九宮坑道，有4處出海口及5座地下坑道聯結碼頭，可以容納52艘小艇停泊。目前免去戰時的恐慌，坑道內放置著模擬蠟像，模擬當時開鑿坑道的情況，讓遊人體會過往的艱辛。參考：九宮坑道 | 金門觀光旅遊網

▲ 以前坑道還兼顧醫院的功能，叫「九官醫院」

▲ 可愛的芋頭寶寶

坐渡輪回金門

▲ 預約了【金豐租車】，更可以選擇在碼頭接我們去租車店，十分貼心

金豐租車

參考：

  1. KmFun金豐租車
  2. 普通 125cc 機車 400元/天
  3. 預約【金豐租車】，可以在碼頭機場/接送去租車店，十分貼心
  4. 【金豐租車】去特約店家消費集點更可以換禮物

▲ 店員雖然看來都是年輕小伙子，但是講解和服務都很好，態度也客氣，所以【金豐租車】是推薦的金門租車店

▲ 機車安全帽有附小護墊，貼心

住宿：金門縣三元民宿 (San Yuan Guest House)

▲ 【三元民宿】是新蓋的民宿，東西都很新，接待人員熱情會推薦行程

▲ 被升級到四人房間，賺到了

▲ 客廳有水果享用

▲ 如果以後退休，希望自己也有這樣一棟房子，只是這事情一定不可能在澳門發生

▲ 雖然沒有供早餐，但民宿有土司和很多點心免費享用

▲ 冰箱飲料都可以拿來喝

▲ 泡麵隨便吃！！！

▲ 每一層的公共區間都有放餅乾點心

參考

  1. 三元民宿 - Google 地圖
  2. 房價：經 booking 預訂，雙人房兩晚 NT$3600，沒有供早餐，但民宿有土司和很多點心免費享用 (2018 / 11)

小吃：蚵嗲之家

▲ 聽說「蚵嗲」已經是金門必吃了，我們在入住民宿後馬上騎車趕到，晚上六點多還是大排長龍

▲ 蚵嗲，30 元，裡面的配料好厲害，再配上醬汁，非常美味，然後我們每天都來買一顆吃吃才過癮

小吃：金門蛋捲

▲ 排蚵嗲的時候在旁邊買一塊閩式燒餅，15 元，蠻好吃

晚餐：米香屋

▲ 牛肉捲餅，75 元，雖然有點小貴，但牛肉是金門的獨有的味道，值得試試

▲ 廣東粥，70 元，經過我這個正港"廣東人"認證，這個粥是不太廣東的，是金門味，不過廣東粥是金門必吃的啦！所以必點

▲ 來台灣一定要喝珍奶，一天一杯就好

▲ 晚上的金城街，紅紅燈籠掛店家

台灣金門三天遊

第一天行程

松山機場 → 金門 → 金城車站 → 水頭碼頭 → 烈嶼（小金門）九宮碼頭 → 九宮港區旅客服務中心租電動車 → 午餐：黃厝三層樓民宿 → 勇士堡/鐵漢堡 + 地雷主題館 → 湖井頭戰史館 → 八達樓子 → 后麟步槍模擬射擊館 → 沙溪堡 → 九宮(四維)坑道 → 坐渡輪回金門 → 金豐租車 → 住宿：金門縣三元民宿 (San Yuan Guest House) → 小吃：蚵嗲之家 → 小吃：金門蛋捲 → 晚餐：米香屋

第二天行程

早餐：永春廣東粥 → 建功嶼 → 得月樓 → 小吃：高樑奶茶 → 金水學校展示館 → 午餐：金道地小吃店-蚵仔麵線 → 明遺老街 → 翟山坑道 → 陳景蘭洋樓 → 八二三戰史館 → 金門迎賓館 → 陳清吉洋樓 → 陽翟老街 → 晚餐：良金牧場工廠總店 → 小吃：蚵嗲之家 → 清金門鎮總兵署 - 夜遊後埔小鎮

第三天行程

退房：金門縣三元民宿 (San Yuan Guest House) → 早餐：聯成廣東粥 → 古寧頭戰史館 → 北山古洋樓 → 北山斷崖 - 北山播音牆 → 慈湖落日觀海平台 → 小吃：三寶齋燒餅店 → 午餐：巧味香 → 小吃：阿公ㄟ豆花 → 金城民防坑道 → 陳金福號 → 莒光樓 → 小吃：蚵嗲之家 → 機場還車
