[旅行] 2019 春 | 東京六天遊 Part01 奇怪飯店的小姐姐你好萌

2019-04-16
2019-tokyo-6days, 2019-東京, http://schemas.google.com/blogger/2008/kind#post, japan, life, lift, tokyo, trip, 旅行, 日本, 東京, 生活

延續去年的【2018 東京六天遊】，在 2019 年的農曆新年也去了東京，主要的行程同樣是去迪士尼樂園和滑雪。

今年去的是「東京海洋迪士尼樂園」，選擇滑雪的地點是「神樂滑雪場」。

這次來回東京的飛機都是半夜或凌晨出發的紅眼班機。沒辦法啦！因為為了省錢也只能這樣，因為過年期間的飛機票很貴的呢！就算我們提前了好幾個月訂的機票價錢也只是剛好能夠接受。

人生苦短呀！既然有機會就要出去玩一下，回來才可以發奮上班賺錢，對不對？

第一晚行程

澳門國際機場 → 台灣桃園機場 → 日本羽田機場 → 飯店：変なホテル東京 羽田 ( Henn na Hotel Tokyo Haneda )

機票

今年同樣是由澳門經台北再轉飛東京，是先吃塊雞排再出發，回程時可以吃牛肉麵的概念。

  1. 澳門 <> 台北 / IT308 2145 - 2325 / IT307 2030 - 2225 / HKD1260 台灣虎航
  2. 台北 > 東京 / MM860 2100 - 0055 / NT$6,310 樂桃 Peach
  3. 東京 > 台北 / IT 217 04:50 - 07:55 / JPY 19,170 台灣虎航

上網卡 - AIS Sim2Fly

這次同樣是用 AIS 的 Sim2Fly 卡，68 RMB 七天 4 GB。這張上網卡是由泰國電訊公司推出的亞洲地區通用漫遊卡，除了日本以外，連台灣都可以用哦！

現在的 AIS 電話卡用完可以繼續用，直接用微信支付充值好方便，但是要每 30 天充值 50B 一次延續壽命

澳門國際機場

大年初一的晚上好誇張，去官也街想買盒葡撻居然賣光光沒貨，最後要跑去威尼斯人裡面的分店才有得賣，而且還要排隊個十幾分鐘呢！兩盒安德魯蛋撻（ 12 隻）要價 130 元。

蛋塔賣得出去，自由行儘量來，澳門發大財。

▲ 去迪士尼當然會有特約娃娃小熊維尼的加入囉

▲ 澳門機場今年的佈置頗用心

▲ 其實機場也有賣葡撻，其實個人認為葡撻要新鮮熱熱的才好吃，外帶坐完飛機之後都冷冰冰了，就失去了葡撻原本的美味

▲ 我千辛萬苦氣沖沖地趕到機場，然後虎航還給我 delay ！！

▲ 看到虎航上販售的台酒泡麵就好想吃，只是價錢貴了好多

台灣桃園機場

▲ 台灣的便利超市非常棒，方便之餘，偶爾逛還會給到驚喜

▲ 經過台灣當然不能錯過手搖杯，現在一航可以買到 COCO 而且沒有加價

▲ COCO 還是台灣的好喝

▲ 又是每次必經的「機場之歌」合照

▲ 樂桃的紅眼班機，乘客都在睡覺，組員幾乎不用做事，好像連降落的檢查都沒有弄（希望是我看錯）

日本羽田機場

▲ 第一次來到羽田機場，感覺果然是比較新穎，走廊高高的好有氣派

▲ 零晨一點，只能先坐計程車去羽田機場附近的飯店休息

▲ 不會講日文沒有關係，Agoda App 裡面有名片功能，打開給司機看就好了

▲ 別看車子小小的，我們 4 個人拉 4 個行李箱都可以坐得下。車子是用電動的好安靜

▲ 只是車內空間就比較擠，計程車上的設備一樣是一堆，一點也不會輸給港澳台

▲ 由羽田機場坐到附近的「変なホテル東京 羽田」( 海茵娜酒店东京羽田 Henn na Hotel Tokyo Haneda ) 只要 10 分鐘左右，計程車收費為 1850Yen，4 個人攤分很可以

飯店：変なホテル東京 羽田 ( Henn na Hotel Tokyo Haneda )

▲ 「変なホテル東京 羽田」( 海茵娜酒店东京羽田 Henn na Hotel Tokyo Haneda ) 是日本近年流行的機械人飯店，在全日本都很多分店，前台的接待人員是兩隻恐龍和一個女生，Check-in 過程是全自動的，不過其實不用擔心，還是會有真人出來幫忙一下的

▲ 機械人不只會說話，還會動的呢！

▲ 還以為揮手會有反應，原來是…..

▲ 話說我們入境日本的時候，海關人員看到我們填在入境單子上的飯店名字是「変なホテル東京 羽田」都忍不住笑了出來，太有趣了

▲ 飯店充滿奇怪的風格，可愛嗎？

▲ 房間空間小是意料中事，可是沒有想到連桌子都是收納式的。好處是可以節省空間啦！就是我們兩個行李箱都可以同時打開

▲ 就是便宜的飯店但是設備也不馬虎，有台小平板查東西和 Handy 手機可以使用

▲ 「変なホテル東京 羽田」( 海茵娜酒店东京羽田 Henn na Hotel Tokyo Haneda ) 果然是新飯店，雖然房間小小的但是很乾淨。牆紙還用富士山
