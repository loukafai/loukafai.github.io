【2019 埃及跟團十天遊】來到第六天。早上醍來，發現游輪已經開到【埃德富】。坐游輪遊埃及真是有趣的觀光方式，順著河流看各處的景點，節省時間之餘又可以看到不同的景色。

▲ 美麗的日出

坐馬車出發

▲ 時間是早上 05：52，大家下游輪之後在碼頭坐上馬車，準備出發。其實十分鐘左右的路程，中間都是水泥馬路，根本不需要坐馬車，一般的巴士就很夠了。只是聽說這樣是觀光方案，讓當地人也能賺點觀光財

▲ 早上只有 17 度，有夠冷

▲ 遇到只會講 "不要 不要" 的馬車司機，好好笑

▲ 這應該是本人生平第一次坐馬車了

▲ 要認得自己的車號，不要上錯車

艾得夫神殿

艾得芙神殿；又作荷魯斯神殿，Temple of Horus）是位於埃及尼羅河西岸城市埃德富的一座古埃及神廟。它是繼卡奈克神廟後最大及保存得最好的一座神廟，除了用作供奉鷹頭神荷魯斯之外，神廟亦作宗教祭祀活動。此外，廟內牆壁的銘文也為學者提供了不少古埃及於希臘羅馬時期在語言、神話及宗教方面的資料。( via 埃德富神廟 - 維基百科，自由的百科全書)


▲ 導遊很有趣準備了一些圖畫來講解，是經典的埃及神話故事 ( via 20190405 艾得夫神殿 - YouTube)

▲ 小插曲：一組懷疑會講中文的遊客跨過欄杆爬上神廟的二樓，被大家罵的時候還裝作聽不懂，最後被大家眾目怒視才願意離開，這個故事告訴我們要當合格的旅人，不要出門丟了國家的臉子

▲ 又拍到我們的游輪 Princess Sarah II

游輪早餐

▲ 果汁好好喝，只有早餐提供。昨天早早起床去看【阿布辛貝雙神殿】，都沒有吃到游輪的早餐，今天終於有個機會給我們吃個夠

▲ 大家都搶著排現做的歐姆蛋，人龍長到都快要半小時吧

跟團友聊著聊著，發現原來竟然有住在中壢的鄰居，說不定之前有曾經見過 XD

▲ 然後把身上的埃磅都一起拿出來拍個照

▲ 埃磅面值 50 的紙鈔有兩種，50p 和 50 Le 不同哦，聽說有小販會騙遊客

▲ 吃完早餐，導遊說要在大廳集合。到達之時，他已經載上一條阿拉伯頭巾

▲ 回房間休息之前，導遊主動說要給我們上一堂，講解一下阿拉伯文化，在他們的世界裡，女人的身份地位真是比較低嗎？ ( via 20190405 講解阿拉伯文化 - YouTube)

▲ 今天的毛巾是法老面具，打開門時嚇了一跳
