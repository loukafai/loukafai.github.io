勇敢者游戲：決戰叢林 Jumanji: Welcome to the Jungle (2017)

via 勇敢者游戲：決戰叢林 (豆瓣)

因為看到這部的續集《勇敢者游戲2：再戰巔峰》(2019) 有好評，所以就把第一集先找來看看。

四個主角因為被罰打掃，一個不小心就被遊戲機吸進去，然後展開逃亡之旅，整個故事很科幻。電影的劇情輕鬆好笑，是一部合家歡電影，動作畫面亦有做到位，喜歡看冒險電影的朋友一定不能錯過。

第一滴血5：最後的血 Rambo: Last Blood (2019)

via 第一滴血5：最後的血 (豆瓣)

講起史泰龍，一定知道是《Rocky》和《Rambo》兩個系列，都是橫跨數十年的經典作品，我個人對於《Rocky》是比較喜歡，因為劇情真的很熱血勵志，而後者《Rambo》則是比較適合喜愛暴力動作電影的朋友。

作為動作電影愛好者，當然不能錯過這個續集，期待這位已經 73 歲的老伯伯還能夠拍出怎樣的電影。

整部電影的劇情是比較簡單的，一句話形容就是老人版的《小鬼當家》，對於男主角開段就被一群小混混痛揍一輪，個人感覺是奇怪，因為以他的身手和經驗真的不應該犯這種錯誤，如果是要說突出後來報仇的劇情，那就有點牽強。

再來，本來是很期待這集的 Rambo 會弄些什麼厲害的機關陷井來對付敵人，Rambo 這次弄的地道迷宮是有點特色，但是沒有到很驚喜的感覺，有待加強。

總結來說，這次的續集仍然是有達到老粉絲的要求，血腥、暴力完全是爽片無誤，希望史泰龍在有生之年仍然要拍下去。

星際探索 Ad Astra (2019)

via 星際探索 (豆瓣)

這部在看之前已經知道不是一般的科幻電影，因為可能許多觀眾都被電影取的標題誤導了，以為是像《星際效應》或《地心引力》那樣戲。

其實，這部《星際探索》是一部科幻文藝電影，沒有太多刺激的動作畫面，也沒有奇怪外星人的入侵，有的是科技進步帶來的種種反思。

電影裡面男主角本來有一段美好的感情，但是因為他為了要去接受太空任務而放棄，就像男主角爸爸一樣，為了尋找外星人丟下男主，然後男主千辛萬苦終於找到爸爸，卻未能成功把他帶回家，看到這一段的感觸最深，因為許多要成就大業的人，總是要把家庭和親人放到最後，很可惜是吧！？

當感受過什麼叫孤獨，才會珍惜眼前人，所以最後男主角發現自己還是喜歡有那種感情牽掛的感覺，重新跑回地球。

個人很喜歡這種電影，因為角色會在劇情發展中得到感悟然後作出改變，不是每個人都可以從一開始做出正確的選擇，我們只需要在適切的時候馬上修正，這樣就很好了。

這部《星際探索》只能推薦給喜歡思考人生的朋友了。

花椒之味 (2019)

via 花椒之味 (豆瓣)

這部是之前聽影評有介紹好看，然後在 FB 上也看到朋友推薦，所以也找來看看，果然不錯，然後也很催淚，如果要看這部記得要準備衛生紙。

每個人在生活上經常需要扮演不同的角色，在公司可能是大老闆，在老婆面前是溫柔的好情人，在子女面前是嚴肅的父母。我們沒有辦法單從其中一面了解到「整個」人，而事實上我們在多數時間只能認識到對方的其中一面，所以會經常產生誤會。

在電影裡面女主角的爸爸（鐘鎮濤飾），他在女主眼中就是只會工作不顧家庭的父親，還在外面有兩個私生女，明顯地，這種的父親是相當不合格，但是後來女主發現，原來爸爸做的火鍋店很受食客歡迎，爸爸對員工很好，甚至最後還發現爸爸的錄音帶是充滿了愛。

所以我覺得，有時候嘗試從多角度去了解身邊的人，也許不是什麼壞事，只是別人會覺得你很八卦就是了 XD

最後，這部《花椒之味》非常適合喜歡慢慢細味的觀眾，裡面許多的對白和場景都有刻意設計，尤其是最後鄭秀文和劉德華最後一段的車上對答，完全擊中許多男朋友的心聲，相信導演和編劇是要對人生很有體會才會拍出這樣出色的作品。

我的事說來話長 俺の話は長い (2019)

via 我的事說來話長 (豆瓣)

這部看劇名十分普通，我是因為看過網友介紹，然後又看到豆瓣有好評才會去追的。

這部《我的事說來話長》劇情就如標題一樣，就是不停看男主角在講話，看他作為一個無業蹲家男如何耍嘴皮，雖然沒有工作還是充滿自信，經常可以振振有辭地講出許多歪理，看他跟親人朋友有趣的互動，相當窩心搞笑，我想就是這部能夠受歡迎的原因。

最後這部推薦給喜歡看日劇的朋友，是一部有別於傳統的刑警醫療日劇風格。

犯罪現場 犯罪現場 (2019)

via 犯罪現場 (豆瓣)

有段時間沒有看香港電影，又剛好看到這部的評分可以，當然要棒場看看。

整部戲主打是懸疑，個人覺得劇情安排都不錯，結局是能夠做到出人意料的感覺，加上有宣萱和古天樂的再次合作，真是給人滿滿的情懷，這點要加分。

最後這部推薦給喜歡看懸疑電影的朋友。

王者天下 キングダム (2019)

via 王者天下 (豆瓣)

這部《王者天下》是由漫畫改編，漫畫在很久以前已經聽過朋友說好看，只是個人一直對這部提不起勁，沒有去追，看到有改編的電影就拿來看看吧！

雖然沒有看過漫畫，只有大概知道相關的劇情，但是仍然覺得這部電影的完成度有達到水準，故事結構都有完整，不會讓人看得一頭霧水，光是這一點就大贏之前的《進擊的巨人》了。

希望這部能夠繼續出個續集。

星球大戰9：天行者崛起 Star Wars: The Rise of Skywalker (2019)

via 星球大戰9：天行者崛起 (豆瓣)

作為半個星戰迷，這部仍然是本年度最期待電影，沒有之一。

這部在看之前已經知道評分有點低，還擔心會不好看，最後還真是「原力與你同在」，這部《星球大戰9》沒有讓我失望。

與其是講電影劇情，倒不如說是電影圓了粉絲們的一個願望，整個《星戰》系列橫跨 42 年，完全能夠說得上是史詩式的巨作，我想喜歡的粉絲一定有愛上它的原因，如果你沒有，講也不會懂。

雖然說是一部經典電影的續集，電影在劇情的設定仍然有經過"進化"，對原力的演譯有更豐富的方式，原力由當初的掐人脖子進化到後期的隔空取物，星戰電影總是能夠給人經典的感覺同時又帶點新意。

說起星戰，當然大家最喜歡的一定是絕地武士光劍，這集裡面的光劍也佔了很多戲份，讓大家不會失望。
