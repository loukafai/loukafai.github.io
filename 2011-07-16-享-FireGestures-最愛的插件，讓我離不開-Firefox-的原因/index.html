我認為 Firefox 上最好用的插件 [ FireGestures ](http://www.xuldev.org/firegestures/)

寫一篇關於 Firefox 裡的一款插件 FireGestures 的介紹 滑鼠手勢 滾輪 使用心得 設定


可以點我安裝

FireGestures 真是一款很好用的滑鼠手勢插件，它的設定很多、功能強大，我用到的功能有：滑鼠手勢（按着右鍵然後上下左右移動滑鼠）、滾輪手勢、搖擺手勢、分頁滾輪手勢。因為有它所以我才一直都在用 Firefox 沒有跳到 Chrome。

以下分享我一些設定心得：

[![Media_httpscreensnapr_lqqcb](http://posterous.com/getfile/files.posterous.com/loukafai/zBJHejjaBdaJGcIqngbuzifrBhsGuCEHqEIlDxHphGAHdBBlAjgmdHqnBJiA/media_httpscreensnapr_lqqCB.png.scaled500.png)](http://posterous.com/getfile/files.posterous.com/loukafai/zBJHejjaBdaJGcIqngbuzifrBhsGuCEHqEIlDxHphGAHdBBlAjgmdHqnBJiA/media_httpscreensnapr_lqqCB.png.scaled1000.png)
▲用滑輪＋右鍵 上下滾動來切換分頁；用先按右鍵再按左鍵關閉分頁。 超級方便！！！
![Media_httpscreensnapr_bfyju](http://posterous.com/getfile/files.posterous.com/loukafai/oCJlGtpdwpFjAypsHIlnuiCFAjbeAlCfsADvzaFuzABxIpttvzcsAmaxAoyA/media_httpscreensnapr_Bfyju.png.scaled500.png)
▲手勢對應，我只有改「上一頁」和「下一頁」
![Media_httpscreensnapr_fazpr](http://posterous.com/getfile/files.posterous.com/loukafai/HfhDrtEsvhmsuonhDGvCesxslCaBgrBFtoejvDqwgHzoenFouHFvqgmAuars/media_httpscreensnapr_fAzpr.png.scaled500.png)
▲跳到頁頂和頁底
![Media_httpscreensnapr_byjgb](http://posterous.com/getfile/files.posterous.com/loukafai/HHbhIEvmAcwwHdkrHmfjzuEHHBAFdarkpcjExvfpIeaAcffczjkEDyewbGgs/media_httpscreensnapr_byjgb.png.scaled500.png)
▲按右鍵再向下拉：全螢幕。用來看我最愛的[Mobile01](http://www.mobile01.com/)長長的開箱文(敗家文)或漫畫都很方便。
