  1. 毒梟 第一季 Narcos Season 1 (2015)
  2. 小心幫忙 A Simple Favor (2018)

我的超豪男友 Crazy Rich Asians (2018)

via 摘金奇緣 (豆瓣)

原本對這部興趣不大，是因為聽了影評說不錯才找來看，看完覺得蠻喜歡，還推薦了給同事。

雖然是荷里活的電影，但是演員和大部分的場景都是來自亞洲，電影被東方觀眾看起來應該親切感滿滿的吧！

戲裡面講好幾個新加坡家族都過著超級侈奢的生活，婚前單身派對在大貨船和獨立小島上辨，非常誇張。其實現實是不是這樣子並不重要，反正電影看得開心就好。

電影這類灰姑娘的故事從來都是吸引，這次更是「升級」版的灰姑娘，擁有經濟學教授資格的女主角就用最後的麻將贏得婆婆的認可，反而電影看不出男主有什麼值得愛的地方，啊！不是，有錢又帥的男人，哪有人不愛？

S風暴 (2016)

via 反貪風暴2 (豆瓣)

原來是想找 2018 的那集，不小心把 2016 的那集找了出來，反正這兩集是前後有關連的，就一起看。

講廉政反貪的電影一直都有固定的捧場觀眾，因為 ICAC 的工作都十分保密，外界很少了解到他們的辦案手法，因此這樣就更能給編劇增加發揮的空間，觀眾入場支持的原因也許是想要知道廉署的秘密故事。

演員找來古天乐、张智霖、周渝民三位型男當主角，光看臉就很爽了是不是？再加上幾段槍戰動作戲，觀眾馬上收貨。

L風暴 (2018)

via 反貪風暴3 (豆瓣)

如果只是拍廉署的案應該是比較沉悶，所以這集拉同警署重案組一起打擊洗黑錢集團，最後中共也加入幫忙，完全正治正確。

看完這部才發現原來在之前還有【Z風暴】(2014)這集還沒補看，有空要再找。

毒液：致命守護者 Venom (2018)

via 毒液：致命守護者 (豆瓣)

沒有蜘蛛印在胸口的毒液就是少了個味。

個人覺得這部【毒液】的特效製作是蠻不錯的，只是好像場景都刻意安排在晚上，不免會給人作弊的錯覺。

還有，劇情講男女主角之間的感情先是分手，之後又莫名其妙的重修舊好，有點讓人覺得好笑。

講到底，高速【毒液】只是用來打發時間的爆米花電影罷了，不用太認真。

鐵血戰士 The Predator (2018)

via 鐵血戰士 (豆瓣)

以前有看過【鐵血戰士】的其他系列，它跟異形的大戰更是必看的經典之作，所以個人對這部【鐵血戰士】還是有點小期待的。

像這種怪物打斗的電影，劇情從來都很簿弱，通常都是莫名其妙就打起來，或是很倒霉的被追殺，這部【鐵血戰士】多了一個元素就是男主角要主動為了報仇去追殺怪物，到最後把怪物幹掉，大快人心。

如果【鐵血戰士】能夠像最近的異形系列【普羅米修斯】那種，多點哲理的內容應該就會好看更多。

小心幫忙 A Simple Favor (2018)

via 一個小忙 (豆瓣)

聽影評有被讚，加上是喜歡的懸疑類型電影，當然要看看。

這部的兩位女主 Anna Kendrick 和 Blake Lively 都是個人喜歡的女演員。Anna Kendrick 電影作品蠻多，最近的【Pitch Perfect】系列算是代表作，但仍然沒有讓她爆紅。這次的【小心幫忙】終於給 Anna Kendrick 可以好好發揮的機會。

關於這部的劇情算是很滿意，整部電影的節奏又剛剛好，劇情具有懸疑性又不會太牽強，值得一看。

不能成為野獸的我們 獣になれない私たち (2018)

via 我們無法成為野獸 (豆瓣)

既然是新垣結衣的日劇，哪有不追的理由？

應該是對上一部【逃避雖可恥但有用】的口碑太好，大家對這部【逃避雖可恥但有用】是滿滿的期待，結果這部得到的評分就偏弱了一些，但是豆瓣上仍然有 7.8 的中度評價。

講回本劇，這部的卡司真是夠大，演員發揮絕對是沒有問題的，失分的原因就只剩劇情。這部是偏向講述職場生活和男女心靈情感的劇，如果觀眾也跟女主一樣，曾經面對過男友出軌、上司給壓力，這部看起來絕對是能夠充滿共鳴。

這部的中心是，“如果可以像野兽一样自由生活就好了。”，就是希望大家可以追隨自己內心的想法，用最適合的方式快樂地生活，中間的過程可能會有點痛苦，但結局一定是令你滿意。

攝影機不要停！ カメラを止めるな！ (2017)

via 攝影機不要停！ (豆瓣)

某天在「字幕組」看到的高分電影，好奇之下找來看看，前半段真的好無聊看到想睡，但是原來電影的有趣之處在後半段。

後半段是講原來前半段的實況電影是如何拍出來，就是戲中拍戲的概念，蠻有趣。

紅海行動 (2018)

via 紅海行動 (豆瓣)

去年票房很好的中國電影，看海報都知道電影類型就是充滿愛國主義的戲，如果是在過年的時候一群人在電影院看一定很感動。電影的戰爭場面和特效真的很棒，這點一定要讚！

不過個人的話會比較喜歡同期的【戰狼】，就是因為有吳京，有一個明確的男主角就夠了，這部主角一堆，就是劇情安排有幾位角色要壯烈犧牲，但是看著也比較無感。

【紅海行動】總的來說是部值得一看的戰爭電影。

月光光心慌慌 Halloween (2018)

via 月光光心慌慌 (豆瓣)

個人本來就很喜歡看驚嚇電影，加上這部是經典電影【Halloween】(1978) 的續篇，所以就找來看看。

這部要講驚嚇也只是一般，整部看起來最爽的就是殺手掉到陷阱的一段了，因為如果被殺手追殺怎麼可能躲回小木屋？結果是準備反擊才比較合理。

希望如果這部有續集可以再嚇人一點。

毒梟 第一季 Narcos Season 1 (2015)

via 毒梟 第一季 (豆瓣)

【毒梟】是一部超高分的美劇，又是講毒品的，就像當年的神劇【絕命毒師 Breaking Bad】一樣受歡迎。

好看的美劇都是需要時間去鋪序，【毒梟】也一樣，開始的前兩集是會普通一點，但是去到第三、四集便開始滲出緊張味道，到後來更會追得停不下來。

覺得【毒梟】這劇精彩的地方是緝毒局對打擊毒品那種鍥而不捨的精神，對面兇悍的毒販也完全毫不畏懼，越打越兇。還有就是反派男主的演技很棒，能夠讓觀眾對他恨之入骨。
