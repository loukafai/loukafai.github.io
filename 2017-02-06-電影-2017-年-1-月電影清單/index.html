  1. 美豬出城

  3. 魔球

  8. 求職記

美豬出城

via 接着侵略哪儿 (2015) (豆瓣)

打個分：

  • 劇情對白 ★★★★★
  • 演員發揮 ★☆☆☆☆
  • 畫面音效 ★☆☆☆☆

一部相當精彩的記錄片，要不是聽到影評介紹還真不會去看。

電影主要是講主角帶著鏡頭去探索世界各個國家的優點，當中有哪些優點是值得美國人去學習。電影一口氣介紹了幾個國家，其中令我深刻的有：

  1. 意大利人的工時很短，每年都有很多休假，但是國家的 GDP 依然很高
  2. 斯洛文尼亞的大學完全免費
  3. 法國學生的午餐超級豐富、而且很有營養
  4. 挪威是採用開放式的監獄
  5. 冰島是個很強調女性地位的國家

看電影，也是吸收知識的方法之一。

叛諜追擊 5

via 谍影重重5 (2016) (豆瓣)

打個分：

  • 劇情對白 ★★★★☆
  • 演員發揮 ★★★☆☆
  • 畫面音效 ★★★★☆

沒辦法，我是叛諜追擊系列的忠實鐵粉，分數自然就要打高一些 XD 但這集的特效是真的做得很出色，導演把以往幾集裡面所有受歡迎的元素都保留起來，街道追車戰、近身搏鬥、綁架，該有的都有了，最後的希望當然是有還會有下一集囉 XD

魔球

via 魔球 Moneyball (2011) (豆瓣)

打個分：

  • 劇情對白 ★★★☆☆
  • 演員發揮 ★★★☆☆
  • 畫面音效 ★★☆☆☆

電影放了好一陣子了，雖然主角是有名的 Brad Pitt，但是當年卻是看到 XDita 一篇推薦的文章才會想看這部戲。

  • 魔球，一個搞清楚產業重點的棒球故事 « Blog.XDite.net

故事是講男主角利用數學方法找出一些不被市場看好的球手，再以低成本組成球隊，但是出來的效果絕佳，最後創出連勝 20 場比賽的歷史紀錄。

主角對自己的決定充滿自信，是最讓人深刻的情節，為了堅持決定的計劃，他甚至還敢去開除教練，而且主角週圍的朋友去勸他也是沒有聽進去，還好最後是真的給他押對，不然也是慘敗收場，一句話「藝高人膽大」就是形容主角這種人吧！一般人還真不能隨便去學。

最記得的一段戲是男主角去教肥子下屬如何去炒掉不被續約的球員，方法就是直接講就好，因為大家都是是專業人士的，真的不用顧慮太多。

這部《魔球》是好戲，有空可以重新再看。

湄公河行動

via 湄公河行动 (2016) (豆瓣)

打個分：

  • 劇情對白 ★★☆☆☆
  • 演員發揮 ★★★☆☆
  • 畫面音效 ★★★★☆

根據真實事件改編，表面上是輯毒戲，但暗地裡是宣揚中國警察的偉大。不過沒關係，電影好看就好。

整部戲就是爽片無誤，看到導演是林超賢基本上就可以放心了，他導的戲一般都能保持水準。這部《湄公河行动》也沒有讓人失望，動作戲一場接一場，海陸空大戰層層推進，完全沒有冷場。

這部《湄公河行动》整體來說十分滿意，唯一不喜歡的點就是最後彭于晏飾演的角色死掉吧！是說就算他沒死，他的身份也蠻尷尬的啦！

柏鳥小姐的童幻世界

via 佩小姐的奇幻城堡 (2016) (豆瓣)

打個分：

  • 劇情對白 ★★☆☆☆
  • 演員發揮 ★★☆☆☆
  • 畫面音效 ★★★☆☆

因為聽影評說這部是兒童版的 X-Man，所以才有興趣看看。

整體故事還算可以，音樂特效都是荷里活水準，只是認為時間停頓(重設)的設定有點多餘，而且吃人眼球的畫面有點噁心，略為兒童不宜。

非常盗2

via 惊天魔盗团2 (2016) (豆瓣)

打個分：

  • 劇情對白 ★★☆☆☆
  • 演員發揮 ★★☆☆☆
  • 畫面音效 ★☆☆☆☆

看過第一集就沒有理由不看第二集，而且這集有部分是在澳門取景，再加上周杰倫和哈利波特的客串，所以就是爛片也要看下去的決心（握拳）。

先說好最好看的一段，就是他們一行人在潛入了機地偷晶片的部分，幾個人互相把撲克牌丟來丟去的一段真的精采又好笑。但是整部片看完結果也是有看沒有懂，看他們隨便一下，彈指之間就能把人催眠，我也是服了。唯獨他們是如何瞬間由美國去到澳門的那個魔術是比較說服人。

這部戲，如果不是粉絲就不要浪費時間了 XD

超人類：卓比

via 超能查派 (2015) (豆瓣)

打個分：

  • 劇情對白 ★★☆☆☆
  • 演員發揮 ★★☆☆☆
  • 畫面音效 ★★★☆☆

很容易被海報騙到的戲，說特效是做得不錯，但是電影到中段有點沉悶，電影主題可能是想探討的人工智慧所帶來的問題，但是最後變成男主角的「死者甦生」，令人摸不著頭腦。

求職記

via 求职记 (2016) (豆瓣)

打個分：

  • 劇情對白 ★☆☆☆☆
  • 演員發揮 ★☆☆☆☆
  • 畫面音效 ★☆☆☆☆

衝著卡司來看的戲，結果是有點小失望，整部戲缺乏主題，完全是為拍而拍，如果 Anna Kendrick 的戲份可以多一點，應該會好一些些。
