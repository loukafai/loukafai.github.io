▲ [生活] 2014 日本京阪神八天遊（8/8）～～京都再會

到最後一天的行程，就是現在遊記越寫也越不捨得，來到日本京都真的給我一種悠閒舒服的感覺，禮貌的日本人、乾淨的街道、生活上到處貼心的小細節，雖然我不確定實際生活在日本是怎麼的情況，但至少我在旅遊方面所體驗到的已經相當滿意。

因為坐樂桃（Peach）廉航回程是早上 7 點的飛機，為了省錢所以選擇了在機場過夜，不過別擔心，晚上的關西機場還有一堆設施可以使用，吃喝玩樂完全不成問題。就懷着放鬆的心情，認真的態度繼續最後的行程吧！

》點這裡看其餘八天的遊記《
