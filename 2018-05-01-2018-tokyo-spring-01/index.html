今年（2018）農曆新年的時候去了日本東京六日，此行的三個重點：1 苗場滑雪、2 東京迪士尼樂園、3 三鷹之森吉卜力美術館

農曆新年期間的日本竟然沒有想像中那麼人擠人，蠻不錯。這次除了正妹之外，同行還有另外四位朋友，我跟他們不算很熟，是第一次跟他們出門旅行，不過大家的默契都不錯。

你知道的，一團人行動總會多了些限制，不像兩個人那麼自由，但是勝在人多的話就能夠取長補缺，大家互相照顧，最後整個行程玩下來都算順利，希望之後還有機會一起出來玩囉！

上網卡 - AIS Sim2Fly

老規矩，先分享這次用的上網卡。

這次是用 AIS 的 Sim2Fly 卡，68 RMB 七天 4 GB。這張卡是由泰國電訊公司推出的亞洲地區通用漫遊卡，除了日本以外，連台灣都可以用哦！

其實之前都用比較便宜的所謂「百度卡」，但是出發前看到 FB 上有朋友分享最近這些卡突然變龜速，加上之前有剛剛去過日本的朋友很推薦這張 AIS 卡，所以安全起見就換了這張試試。

心得：速度快，即插即用好方便，令我意外的是有時候在東京地鐵，同行正妹用的中華漫遊會偶然沒訊號或降速，但是我的 AIS 卡又不會，好神奇。下次可以再考慮用這張。

  • 淘寶 - AIS sim2fly 亚洲14国上网卡 日本4G/2G无限流量手机卡电话卡 - 飞猪

機票

之前的遊記都沒有特地寫，這次開始要記一下，給個參考。

  1. MFM - TPE - NRT / IT 8308 / 00:20­ - 01:55 / IT 200 / 06:35­ - 10:30 / 虎航含 15kg 行李和 Tigerconnect / HKD 2,931
  2. NRT - TPE / GK11 / 21:15 - 00:30 / 捷星含行李 15kg / ¥18670

凡是農曆新年出發日本的機票，在澳門或香港出發都一定漲價不少，所以一定！一定！一定要提早訂票，最後就選擇了搭虎航在台北轉機，在台灣下飛機時會有地勤小姐來接機，主要是叫你乖乖在機場的轉機區待著，不要亂跑。

機票內的 Tigerconnect 服務是強迫要加購，這樣就可以行李由澳門直掛東京，不用重新辨一次出入境。

到桃園機場時地勤小姐還要求幫我拍一張照片，說萬一找不到我會方便傳給同仁一起找，我當然知道一切都是借口，原因只有一個，因為我太帥！哈哈

桃園機場

在機場睡了一晚，先在機場的候機室來一碗牛肉麵，「大豐的 CTM Visa Signature 信用卡」真棒！現在申請會送二年免費無限次使用候機室，二年之後怎麼辦？再看看囉。

成田機場 - 第二航廈

虎航由台北到成田是降落在第二航廈，這點真的很棒，因為如果在三航的話還要再坐巴士才能轉地鐵。

入境之後我們就在成田機場直接買一張 JR東京廣域周遊券 ，10,000yen，而且即時劃好了後天去【越後湯沢駅】的指定席。

機場 JR 車站的售票員會說英文，但是如果會點日文的話會更方便，這個真的很謝謝會正妹，很多需要用到日文的時候都由她出馬（強！）

這張 JR TOKYO Wide Pass 可以先買然後指定開始使用的日子，連續三天之內不限次數坐 JR 好方便，而我們主要是使用它由【東京】坐新幹線去【越後湯沢駅】，這樣就省很多錢和時間哦！

▲ JR東京廣域周遊券

▲ 在買票同時就即時劃好了後天去【越後湯沢駅】的新幹線指定席

坐比較便宜的 成田 skyaccess 到【日暮里站】，再買一張 JR 地鐵市區一日卷，780Yen。

不過我們試了幾台機器都沒有買到，問了服務員才知道這個卷要找綠綠的機器才能買。

飯店：APA 飯店 - 巢鴨站前 (APA Hotel Sugamo Ekimae)

這次的東京之旅在選擇飯店的部分，全部都聽取了正妹的建議住 APA ，重點是 APA 的飯店都有浴場，適合每天都會走很多路、行程滿檔的我們。

飯店就在巢鴨站出來走 2 分鐘左右，交通方便，地鐵【三田線】和【山手線】都有到，

飯店週邊還有Summit 超市、巢鴨商店街，餐廳也蠻多的，機能很方便。

這次的東京之旅住了兩家不同的 APA，發現它們的風格都很統一，大堂弄掛了個大大的水晶燈，弄得蠻有氣勢的。

▲ 房間小小的，二個行李箱同時打開就沒有空間走路了

入住的兩晚都有去飯店的大澡堂泡澡和洗澡，很神奇！就是在迪士尼玩了一整天之後，回來泡一泡之後隔天真的迅速恢復體力，我想以後來日本玩都會盡量挑有大澡堂的飯店來住。

可能很多人都不習慣像日本人這樣脫光光的一起洗澡，會害羞，我一開始也是啦！不過其實洗澡的時候眼睛都不會四處看，專注洗自己的就好，我這次洗到一半還有個阿姨跑進來打掃呢！但是大家都好像見怪不怪，一臉淡定的樣子，好厲害。

因為澡堂不能拍照，就借用一下官網的照片。APA 的澡堂都設在飯店頂樓，當中更弄了一個露天風呂，抬頭都可以看到星星月亮，十分有情調。

▲ アパホテル〈巣鴨駅前〉スパ・大浴場｜【公式】アパホテル（アパ直なら、最安値）｜ビジネスホテル予約サイト

午餐：元祖 塩大福 みずの

因為出發之前剛好看到一直 follow 的部落客介紹，說在巢鴨商店街的【塩大福】是個在地的特色，所以放好行李就直接跑來吃午餐。

原來我對【塩大福】的受歡迎程度是充滿疑問的，但是在店家門口看到排隊人潮，而且沒有遊客都是日本人，就放心了。

【塩大福】是外皮沾了鹽巴，內餡用紅豆的大福，有白色和綠色的，同行夥伴一致認為綠色的艾草味比較好吃，但是我作為麻糬的腦殘粉，我當然是認為......

兩種口味都好吃！！ XD

除了【塩大福】之外，店裡還有拉麵、丼飯可以點。

▲ 260Yen 兩顆

▲ 叉燒拉麵加麻糬 670Yen，味道適中，叉燒厚厚一片好感動

▲ 老店員可能是聽錯我們的單，也可能是我們表達不準，點錯了，拉麵變成了擔擔麵，一樣是 670Yen，但是味道好普通，就是一般的蔬菜麵。不過看到隔桌的日本也有點這個，難道日本也蠻喜歡這口味？

▲ 同樣是 260Yen，可以換成干貝的組合

巢鴨商店街

因為是星期日的下午，人潮比較多。【巢鴨商店街】是有種比較懷舊市場的感覺，都賣很多傳統的東西。

連賣餅乾的店家都充滿一種當代味。

原本以為三度醬油的干貝已經很夠咸了，誰不知還有十度醬油的，黑得跟木碳一樣，一定很咸。

▲ 天氣真好，走在路上很舒服

秋葉原

作為宅男，來到東京就一定要逛秋葉原。

原來秋葉原在星期日下午都會把馬路設成行人專用，方面大家逛，這種政策真的好貼心。

同行夥伴指定要逛的扭蛋店，真的好多扭蛋機，什麼都有的扭。

更令我沒想到的是連搞笑藝人和學校椅桌都可以是扭蛋。

點心：築地銀だこ 章魚燒

冷冷的天氣看到有章魚燒讓人受不了，馬上排隊。章魚燒有三種口味可以選，但是正妹卻說選原味的就好 Orz 因為比較便宜。

旁邊有一家在賣烤肉串，好香，讓我想起之前在白村鄉吃的那串飛驒牛（口水）。

吃完點心就去逛【驚安殿堂・唐吉訶德】已經算是既定行程，第一天就敗家。

就在快結好帳的時候想起有【2018「激安」優惠券！出示手機版折扣券 驚安之殿堂購物即減￥2,000】優惠，但是店員卻說不行！要將優惠卷印出來才算，我們就反駁說網頁上明明就有寫直接刷手機上的 code 也可以，然後店員就說已經結完了不能再刷優惠。

這是我們第一次在日本碰到不爽的購物經驗。

▲ 遇到很熱門的馬里奧卡丁車

HARBS - ハーブス上野店

正妹說自從之前吃過 HARBS 的水果千層蛋糕就一發不可收拾，念念不忘！所以 HARBS 早已經被列入必到行程。

我們趕在 8 點關店前趕到，招牌的水果千層派當然早已賣光，但是既然一場來到當然不能空手而回，它們家別種口味的蛋糕也是很好吃的。

此行一共吃了三次的 HARBS 蛋糕，好誇張。

▲ 就買了三顆大家一起分著吃

▲ 看到美味的 HARBS 蛋糕，減肥運動什麼的完全忘得一乾二淨

晚餐：天丼てんや 上野店

  • 平價天丼 天屋 - Google 地圖

晚上在【阿美橫丁】吃晚餐，大家都不知道要吃什麼，所以我就隨便打開 google map 找了這家【天丼てんや】，因為看分數和評論都還可以就試試。

到了店家門口看到裡面人蠻多的，應該不會是地雷。

超幸運的我們碰到了每月 18 都有的推廣活動！當然是毫不猶豫的點打拆的兩款 XD

夥伴們都對這家【天丼てんや】很滿意，便宜好吃。

▲ 550Yen 同時吃到炸蝦丼飯和蕎麥麵

▲ 390Yen 的天丼，好便宜，而且味道不錯。

阿美橫丁

原來打算吃飽逛逛【阿美橫丁】，誰知道很多店鋪都已經關了，只好下次再來逛。
