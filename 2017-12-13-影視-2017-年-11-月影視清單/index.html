2017 年 11 月影視清單，個人觀影後的排名如下：

  1. 四重奏 カルテット (2017)
  2. 天才槍手 Bad Genius (2017)
  3. 極盜車神 Baby Driver (2017)
  4. 亡命天涯 The Fugitive (1993)
  5. 安娜貝爾2：誕生 Annabelle: Creation (2017)
  6. 變形金剛5：最後的騎士 Transformers: The Last Knight (2017)

變形金剛5：最後的騎士 Transformers: The Last Knight (2017)

via 變形金剛5：最後的騎士 Transformers: The Last Knight (2017)

這部《變形金剛5》讓我分開看了 4 次才能看完，因為實在是太無聊了，覺得如果這個《變形金剛》系列的下一集沒有突破性的劇情，應該可以不用再拍下去了。不過我仍然是會繼續看的 XD

《變形金剛》明明有龐大的預算，理應不難拍出一部出色的電影，但不懂導演 Michael Bay 最後為何會拍得這麼爛，而且不是只有我一個人覺得，是在豆瓣上的評分（4.8）也能夠反映出來。

覺得《變形金剛》會這麼的不討好，其中一個原因是觀眾對變形機械人的新鮮感已經減退，也許剩下唯一的出路是跟別的電影來個 crossover，例如《變形金剛 VS 哥斯拉》或《變形金剛 VS 蜘蛛俠》會是不錯的選擇。

極盜車神 Baby Driver (2017)

via 極盜車神 Baby Driver (2017)

這部是由同樣愛看電影的朋友介紹，二話不說便立即找看看，原來真的不錯。

光看電影名字《Baby Driver》是不會知道演什麼的，但是只要由電影開始看不到 10 分鐘，就會被首幕的飆車戲份吸引到，然後就一直看下去。電影很簡單，三個元素：飆車、音樂、愛情，總之就是爽片一部，看了絕對不會後悔。

安娜貝爾2：誕生 Annabelle: Creation (2017)

via 安娜貝爾2：誕生 Annabelle: Creation (2017)

特地找中午時候才來看這部戲，因為看評論這部戲很恐怖，那支娃娃把每個人都嚇嚇叫。

結果在中午看這部《安娜貝爾》的決定是正確，不然在晚上看真的會嚇死人。不過這部的劇情真的很無腦，就是一隻娃娃對人類充滿了怨恨然後不停的殺人，到最後竟然還殺不死，繼續找別人來殺。

所以如果只想找部恐怖電影來嚇嚇自己，劇情不是重點的話就看這部《安娜貝爾》吧。

電影演到最後是講娃娃被搬走，到了新「家」之後就繼續開殺，劇情上就剛好接上第一集的開場一段殺鄰居的戲。對上一集是 2014 的電影，不過劇情當然已經忘光了，有興趣就重新回來回顧一下囉！

亡命天涯 The Fugitive (1993)

via 亡命天涯 The Fugitive (1993)

這部是因為之前在豆瓣上看到是經典必看電影，所以便留著來看，發現這部雖然是 1993 年的電影，但是即便放在現在來看，依然是毫不遜色。甚至最近由吳宇森拍的那部《追捕 Man Hunt》亦頗有「致敬」的意味！之後應該會找來看看比較一下。

講回這部《亡命天涯》，是講職業為醫生的男主角一直在逃命，到最後還反客為主找出兇手，為自己討回公道，真的有夠厲害！說明要當醫生是很需要聰明的頭腦。

四重奏 カルテット (2017)

via 四重奏 カルテット (2017)

一天看到新聞 - 《2017爆紅女優Top 10》今年最活躍的日本女演員究竟有哪些？，吉岡里帆拿到了第一名，於是便找來這部她有份演出的日劇《四重奏》來看看介紹，看完評論之後覺得可以這部日劇也可以追看看。

個人現在已經很少會故亂去追日劇，因為好看的日劇已經越來越少，多半都是看完評論或有人推薦時才會追看，而這部《四重奏》在豆瓣上有到好評，又剛好想看看所謂的爆紅女星到底有什麼吸引之處，所以便打開來看看，結果一看到尾，沒幾天就把這部看完了 XD

喜歡這部日劇，以風景漂亮的輕井澤作為背景，然後日劇裡面充滿了大量耐人尋味的台語，應該是比較有人生歷練的人才寫得出來。

《四重奏》裡四位主角，各人背後都懷著不堪回首的過去，但是因為一件人口失縱事件聚在一起，美其名是組成樂團，背後卻是另有目的，只不過在劇情發展途中四人又互相扶持，到最後疑團解開終於真相大白。愛情、友情、親情，也許不過是自己的一相情願，能夠有緣相聚就共奏一曲吧！

最後再提一下吉岡里帆，她在《四重奏》中表現亮眼，即使放到松隆子、高橋一生等大咖面前也不會失色，難怪之後她演出的廣告會一部接一部不是沒有理由的。

天才槍手 Bad Genius (2017)

via 天才槍手 (2017)

一部大獲好評的泰國電影，內容就是講學生作弊的故事。

原來以為電影充滿了很多神奇好玩的作弊方法為觀眾帶來驚喜，不過電影裡面並沒有，反而主要刺激好看的部分是電影的節奏感和厲害的剪接技巧，特別是在最後一段講主角去澳洲做「大生意」，真是緊張得連小心臟都快要跳出來。

這部雖然是以作弊為題材的電影，不過到最後的結局當然是要拍得比較正面一點，做人當然是要對得起自己的良心，所以個人十分認同最後女主角的選擇把真相公開，這樣就不用怕被人抓住把柄一輩子啊！
