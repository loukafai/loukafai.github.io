[旅行] 台灣金門三天遊~~Day03

2019-04-16
2018-kinmen-3days, 2018-金門, life, lift, taiwan, trip, 台灣, 旅行, 生活, 金門

金門之旅第三天。

今天逛到有特色的【北山播音牆】，還有講解很精彩的【莒光樓】，最後更買了很有金門特色的蚵嗲和燒餅回家。

可以不誇張的說，金門是個人認為台灣最好玩的地方，因為每處景點都充滿了歷史的痕跡，在參觀的同時會認識到過去曾經有數百萬人為了和平付出了相當多的努力，透過講解人員的口中，會了解到更多金門的故事。

所以說，金門值得一來。

第三天行程

退房：金門縣三元民宿 (San Yuan Guest House) → 早餐：聯成廣東粥 → 古寧頭戰史館 → 北山古洋樓 → 北山斷崖 - 北山播音牆 → 慈湖落日觀海平台 → 小吃：三寶齋燒餅店 → 午餐：巧味香 → 小吃：阿公ㄟ豆花 → 金城民防坑道 → 陳金福號 → 莒光樓 → 小吃：蚵嗲之家 → 機場還車

退房：金門縣三元民宿 (San Yuan Guest House)

▲ 【三元民宿】值得推薦

早餐：聯成廣東粥

▲ 大大的牆面給大家簽名

▲ 海鮮粥 70 元，油條 10 元。私心覺得【永春廣東粥】比較好吃

古寧頭戰史館

西元1949年12月24日，發生著名的「古寧頭戰役」，戰爭結束後，也開始了國、共雙方的長期對峙。1979年雙方砲擊止後，金門軍方便在古寧頭興建「古寧頭戰史館」。參考： 古寧頭戰史館 | 金門觀光旅遊網

外觀仿古城堡式的戰史館，兩旁擺置著當年參加古寧頭戰役的「金門之熊」M5A1戰車，室內陳列戰利品武器、作戰文件、作戰指揮官照片、戰情大型油畫展示…等，為近代重大歷史的戰役紀錄下當時的場景。

▲ 有影片看，我們也在這聽了講解

▲ 「古寧頭戰役」是國軍唯一打贏的戰役

▲ 很特別的油畫，從左右兩邊看會有不同的效果

▲ 另一張會改變方向的油畫

▲ 因為「古寧頭戰役」才打了七十幾個小時，所以後來透過士兵的口述把當時的情況畫了出來

▲ 共國被打敗

▲ 李光前將軍帶頭進攻，最後犧牲

▲ 這種槍炮成為了戰勝功臣

▲ 沙灘上一整排的防登陸樁

北山古洋樓

如果要選一棟最具代表金門過去長期飽受戰火摧殘，所留下的歷史傷痕，那無疑就是建於民國17年的「北山古洋樓」。它經歷了古寧頭大戰、823砲戰加上「單打雙不打」等數不清的砲彈攻勢。參考：北山古洋樓 | 金門觀光旅遊網

▲ 當年的彈孔還留著

▲ 洋樓現在改為民宿，要是膽子夠大的人都可以去住住看

北山斷崖 - 北山播音牆

北山播音牆，「親愛的大陸同胞你們好，我是鄧麗君．．．．．．」鄧麗君那清脆的嗓音從巨大的播音器裡傳出，位於北山斷崖上的北山播音牆，與馬山觀測所的馬山播音站，同為金門戰地時期用來喊話打「心理戰」的前線，用國語、閩南語廣播新聞、宣傳、評論、招降等訊息。參考：北山播音牆 | 金門觀光旅遊網

▲ 剛好遇到有播音，錄到一小段

▲ 控制室

慈湖落日觀海平台

如果請問金門人欣賞落日的好去處，慈湖絕對能榮登在地首選！慈湖位在金寧鄉，1969年為了戰略考量而興建的慈堤，隔在堤外的是海域，在堤防內的部份就是現在的慈湖了，後來再增建了觀景平台，讓慈湖美景更能受到外來遊客的歡迎。 參考：慈湖 | 金門觀光旅遊網

▲ 好多家「金合利」，都不知道哪家才是真的

小吃：三寶齋燒餅店

▲ 還說賣完為止那麼誇張？

▲ 買兩塊三寶齋燒餅試吃一下，鹹甜口味都好吃！準備下午再帶盒回家 XD

午餐：巧味香

▲ 米粉 40 元，蚵仔麵線 80 元，綜合小菜 50 元。味道可以，但是勝在地點方便。

▲ 經過好多次，終於有一張合照

小吃：阿公ㄟ豆花

▲ 豆花忘了拍，都很好吃

金城民防坑道

自1949年之後，戰地的金門，為達到「保存戰力於地下，發揚火力於地上」的戰術目標，從1968年開始，利用金門花崗岩的地質條件，大量構築地下坑道，金城民防坑道即其中之一。參考：金城民防坑道 | 金門觀光旅遊網

▲ 看完影片再出發，坑道走一程要三十分鐘，蠻久的，但是也當作一種體驗

如今戰事已經過去，金城民防坑道則成為頗具教育意義的戰時遺跡，每日皆有免費導覽，讓參訪的旅人更深入瞭解征戰時期的驚惶

陳金福號

▲ 著名的金門貢糖，選它是因為【金豐租車】有合作集點 XD

▲ 沒有想到貢糖有那麼多口味

▲ 很多口味可以試吃，特別是豬腳貢糖很不錯

▲ 服務人員還跟我們介紹制作貢糖的過程

莒光樓

面臨金城美景，背擁豐蓮山脈，初訪金門，必定要登上莒光樓遠眺金門風光！莒光樓做為金門的精神象徵，以中國傳統建築的作法，將傳統城樓放在現代建築的基座之上，締造一項獨特建築美感。參考：莒光樓 | 金門觀光旅遊網

這座建築是為了表彰在古寧頭戰役中的國軍官兵而建，也傳達「固若金湯，雄鎮海門」的戰地精神，成為金門征戰時期最具代表性的地標

▲ 我們又在【莒光樓】聽了好有趣生動的講解，非常謝謝這邊的解說人員

▲ 幸福金門

小吃：蚵嗲之家

▲ 坐飛機前又來拼蚵嗲，又是要排隊呀！！

▲ 好大瓶高樑 XD

機場還車

▲ 在【良金牧場工廠總店】和【陳金福號】敗家太多集了 15 點，可以在【金豐租車】換到 100 元現金和明信片一份

▲ 男的風獅爺

▲ 女的風獅爺

▲ 我們不是坐這台飛機，可能因為這班次的人太多，飛機直接升級成 A320 好開心

▲ 跟地面工作人員說 88

▲ 最後以漂亮的日落來結束三天二夜的金門之旅

台灣金門三天遊

第一天行程

松山機場 → 金門 → 金城車站 → 水頭碼頭 → 烈嶼（小金門）九宮碼頭 → 九宮港區旅客服務中心租電動車 → 午餐：黃厝三層樓民宿 → 勇士堡/鐵漢堡 + 地雷主題館 → 湖井頭戰史館 → 八達樓子 → 后麟步槍模擬射擊館 → 沙溪堡 → 九宮(四維)坑道 → 坐渡輪回金門 → 金豐租車 → 住宿：金門縣三元民宿 (San Yuan Guest House) → 小吃：蚵嗲之家 → 小吃：金門蛋捲 → 晚餐：米香屋

第二天行程

早餐：永春廣東粥 → 建功嶼 → 得月樓 → 小吃：高樑奶茶 → 金水學校展示館 → 午餐：金道地小吃店-蚵仔麵線 → 明遺老街 → 翟山坑道 → 陳景蘭洋樓 → 八二三戰史館 → 金門迎賓館 → 陳清吉洋樓 → 陽翟老街 → 晚餐：良金牧場工廠總店 → 小吃：蚵嗲之家 → 清金門鎮總兵署 - 夜遊後埔小鎮

第三天行程

退房：金門縣三元民宿 (San Yuan Guest House) → 早餐：聯成廣東粥 → 古寧頭戰史館 → 北山古洋樓 → 北山斷崖 - 北山播音牆 → 慈湖落日觀海平台 → 小吃：三寶齋燒餅店 → 午餐：巧味香 → 小吃：阿公ㄟ豆花 → 金城民防坑道 → 陳金福號 → 莒光樓 → 小吃：蚵嗲之家 → 機場還車
