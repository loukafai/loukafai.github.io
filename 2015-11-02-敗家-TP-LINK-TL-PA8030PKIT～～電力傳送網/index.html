[敗家] TP-LINK TL-PA8030PKIT～～電力傳送網

用久了無線網路，就會發現無線訊號的不穩定，速度時快時慢，要拉線是不太可能（也懶的拉），近年的電力 HomePlug 技術已經進步不少，售價也親民，就趁著升級電腦的同時換個 HomePlug 試試看。

價錢～～HKD 780

  • TP-LINK TL-PA8030PKIT - 網絡器材 - 網絡 - 電腦 - 香港格價網 Price.com.hk
  • TP-LINK TL-PA8010PKIT - 網絡器材 - 網絡 - 電腦 - 香港格價網 Price.com.hk

3 port 只比 1 port 版本貴 HKD 50 左右，直接選 3 port 的不用煩

TP-LINK TL-PA8030PKIT 特色

  1. 每部設有3個Gigabit LAN Port
  2. 極速1200Mbps AV2 2x2 MIMO w/Beamforming
  3. 配備電源插座 可供電其他設備
  4. 電源濾波器 減低電子雜訊干擾

更多資料在這邊 >>>> TP-LINK最新超極速AV1200 AV2 MIMO Gigabit HomePlug Kit w/ 3xLAN Port + AC Passthrough - TL-PA8030P-KIT●1200Mbps●3xGiga LAN●配備電源插座●4K TV高清串流

▲ 實際使用的情況，左邊是插在拖板上，右邊是直插電座上。因為其中一邊是接了拖板，所以測出來的速度有差一些

▲ 內網的測試速度使用 iPerf，最快大概是 120 Mbits/sec，以 Gigabit LAN 來說是有點強差人意，但勝在使用有線網路會比較穩定。

▲ 至於 internet，家裡是用 CTM 100M 的光纖，之前使用無線網路時速度只能用到十幾，升級有線網路之後就可以完整用到 100M 了，Yeah！！
