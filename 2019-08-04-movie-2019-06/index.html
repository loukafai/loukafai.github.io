2019 年 6 月影視清單，個人觀影後的排名如下：

  1. 我要准時下班 (2019)
  2. 大偵探皮卡丘 Pokemon Detective Pikachu (2019)
  3. 驚奇隊長 Captain Marvel (2019)
  4. 生死時速 Speed (1994)
  5. 湮滅 Annihilation (2018)

湮滅 Annihilation (2018)

via 湮滅 (豆瓣)

一部評分很高的科幻電影，亮點是 Natalie Portman 和 NETFLIX，記得這部在剛剛上架的時候就立即注意到，結果等到今年才拿出來看看，不看白不看，看完之後還是摸摸頭，有看沒有懂。

建議是除非對科幻電影很有愛，擁有超強的理解能力，否則還是不要隨便挑戰這部。

生死時速 Speed (1994)

via 生死時速 (豆瓣)

非常經典的電影，說是奇洛李維斯的成名作，因為最新的《John Wick》第三集讓他又紅了一把，突然想到所以特地找回來看看。

1994 年的電影，許多特技場面當然不能和今日比較，但是以二十多年前來講應該是超水準，就像當年的《二十世紀殺人網絡》也是神一樣的大作。

大偵探皮卡丘 Pokemon Detective Pikachu (2019)

via 大偵探皮卡丘 (豆瓣)

動畫在我小時候每天都看，遊戲在當年也是超級著迷，真人版電影終於在今年看到，等得實在有夠久，所以對這部十分期待。

這次真人電影的故事是以偵探為主題，雖然是跟日本原來的故事有差異，但是仍然符合原著裡面那種"帶著小精靈"冒險的格調，只是登場的小精靈不夠多，但是粉絲看到會講話的皮卡丘在大螢幕跑來跑去應該就很開心了。

我要准時下班 (2019)

via 我，到點下班 (豆瓣)

可能是因為講職場故事的日劇，所以在本季有點小火熱，就像當年的《半澤直樹》，更應該是說，上班族都愛看日劇吧！就是覺得每天下班之後能回家坐在沙發上看看日劇、吃零食就是所謂的"小確幸"了。

講回日劇，女主角是很久沒見的吉高，以前在【神探伽利略 第二季】就在看她了，果然人生閱歷多了，演技才會有成長。這次夥拍帥哥向井理，可以預料到劇情是除了職場上班，還會有男女感情戲。

《我要准時下班》就是要給觀眾們反思，人生缺缺數十年，除了工作外，其實還有更多的人、事、物值得我們去珍惜，可能是因為日本的加班問題太嚴重，所以才會催生出這部日劇。

劇情當中有一個很有趣的論點，就是說平常看到的那些沒天沒夜加班的工作狂，別以為他們真的是責任心很重，其實，他們只是除了工作以外「一無是處」的可憐人，因為他們不懂享受生活，所以只能把生活重心放在工作。如果從這個方面去想，那些加班狂好像也不是那麼值得敬佩。

這部絕對要推薦給工作狂，以及"迫使"下屬加班的工作狂主管。

驚奇隊長 Captain Marvel (2019)

via 驚奇隊長 (豆瓣)

這部在剛上映的時候就已經聽到被差評了，所以就沒有進電影院看的 feel，直到最近影片上架，終於有機會看看，確實很一般呢！

這部比較多是被批評主角不夠漂亮，又有說是故事無趣，但我個人覺得最大的原因是主角太強，boos 太弱，這樣觀眾就沒有跟主角一起對抗大壞蛋的共鳴感，電影就少了一個味。

對於漫威粉絲，這部仍然是不能錯過的，我們只好希望第二集會拍精彩回來囉！
