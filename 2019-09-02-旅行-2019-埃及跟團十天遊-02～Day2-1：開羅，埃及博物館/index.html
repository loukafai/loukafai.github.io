[旅行] 2019 埃及跟團十天遊 02～Day2-1：開羅，埃及博物館

2019-09-02
2019-egypt-10days, 2019-埃及, egypt, http://schemas.google.com/blogger/2008/kind#post, lift, trip, 埃及, 旅行, 生活, 非洲

我們在飛機上過了一夜，早上約 8 時到達開羅，然後隨即安排第一個景點～～埃及博物館

我在這次的【2019 埃及跟團十天遊】都一直把導遊的介紹拍下來，照片是死的，有聽過導遊的介紹才是真的的來過埃及。

第二天行程

開羅機場 → 埃及博物館 → 午餐：Andrea Memphis Maadi → 穆罕默德阿里清真寺 → 薩拉丁城堡 → 晚餐：Citadel View Alain Le Notre → 飯店：Grand Nile Tower → 飯店兌換埃磅 → 飯店康樂設施

開羅機場

埃及開羅是舉世聞名的旅遊城市，果然機場弄得很有氣派。

旅行社有在機場安排了接待人員，在我們剛剛下飛機還沒出境的禁區範團，就馬上來服務大家，幫大家弄簽證貼紙什麼的。

在這裡不得不吐嘈一下，原來澳門人用特區護照進入埃及不用簽證，台灣人則要買 25 美金的貼紙（團費有包含），那個埃及接待人員聽到我是澳門人就馬上把貼紙（＄＄？）自己收回去。

別看我眼睛好像有個黑眼圈，其實我是很精神的，看我拿到領隊派發的耳機，就像個特務的樣子。

順利出境之後，領隊給大家一點時間去換埃磅，也可以兵分二路去買電話卡，當然，如果買不到的話，領隊有安排助理幫大家代買電話卡，一張 20 美金的樣子

▲ 如果想省錢的話，可以找機會自己去買手機卡，好像會便宜很多，我們是沒有要買，只是好奇順手拍了一張價目表

▲ 還有 "橘子" 電訊商可以選 XD

▲ 小插曲，話說大家都衝去機場唯一一家的兌換店換埃磅，又剛好店家的電腦停電了，說要等大概 20 分鐘才能再換，沒有換到錢的團友只好放棄

▲ 坐上遊覽車，這個 "短髮" 的帥哥就是我們這幾天的導遊「MOMO」，會講流利中文，景點講解詳細，講話又很幽默，是個好導遊，同行還有一個小哥助理幫忙打點東西

▲ 沙漠國家溫差好大，別看早上外面太陽那麼大，其實很冷

▲ 開羅的地鐵，聽說很便宜

▲ 難得來到埃及，相機當然要按個不停

因為這次是難得的埃及之旅，所以在行程中都錄了影片留給以後回味，這是導遊的介紹。

那爾邁調色盤 Narmer Paletter

那爾邁石板是一件非常重要的埃及考古發現，測定年代為公元前31世紀，含有至今為止發現的最早象形文字銘文。它被認為是用來描述那爾邁國王統一上下埃及的。調色板分為兩面，一面描述戴著上埃及王冠的那爾邁，王冠為球根狀；另一面描繪了戴著下埃及王冠的那爾邁，王冠為平頭狀呈紅色。這件文物和同樣在希拉孔波利斯發現的蠍子權杖頭和那爾邁權杖頭一起，提供了對埃及國王的最早描述之一。調色板顯示了許多埃及藝術的傳統手法。埃及古物學者鮑勃·布萊爾（Bob Brier）把那爾邁調色板評論為「世界上第一件歷史文檔」。 via 那爾邁調色板 - 維基百科，自由的百科全書
