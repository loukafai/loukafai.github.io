2017 年 10 月影視清單，個人觀影後的排名如下：

  1. 我只是個計程車司機 A Taxi Driver (2017)
  2. 老手 (2015)

  5. 黑魔塔 The Dark Tower (2017)
  6. 只有我不在的街道 僕だけがいない街 (2016)
  7. 異形魔怪 5：血脈 Tremors 5: Bloodline (2015)

蜂鳥特攻 Hummingbird (2013)

via 蜂鳥特攻 (豆瓣)

之前衝著電影封面的 Jason Statham 就收藏了這部戲，之後就一直放著，趁著有空就把這部清掉。

看完的心得就是一部很普通的動作電影，但是在豆瓣也有 6.2 分，算是中規中矩啦！

故事也蠻平凡的，就是男女主角的設定比較特別，退役士兵遇上善良修女，最後竟然搭上了，只是最後的結局沒有安排他們在一起，頗符合老外的電影風格。

蜘蛛俠：強勢回歸 Spider-Man: Homecoming (2017)

via 蜘蛛俠：英雄歸來 (豆瓣)

記得《蜘蛛俠》電影系列的上一集 The Amazing Spider-Man 2 (2014) 最後的部分 Stacy （ Emma Stone 飾）死掉了，誰料到結果整個系列都重啟了。

這集是 《蜘蛛俠》 重新回到 Marvel Studio 懷抱的一集，整部電影的拍攝手法變得更活潑，而且更可以加入鋼鐵人客串，發揮協同效應。

個人對於這次的《蜘蛛俠》十分滿意，尤其是男主角竟然喜歡上反派敵人的女兒這部分，充滿戲劇衝突。

老手 (2015)

via 老手 (豆瓣)

韓國電影一般不是我的首選，但就是衝著豆瓣 7.5 分才看這部，不知道是不是被分數先入為主而影響到，覺得這部《老手》真的蠻好看呢！

《老手》是描述一個見義勇為的刑警，不畏強權地為受害者追查真相的故事，電影的節奏明快，過程中連場打鬥，十分精彩。

作為一名困難群眾，我想很多人都像我一樣，在遇到困難的時候都希望會有人出手相助，但實際上真的有嗎？應該少之又少，所以這部《老手》

我只是個計程車司機 A Taxi Driver (2017)

via 我只是個計程車司機 A Taxi Driver @movies【開眼電影網】http://www.atmovies.com.tw

是最近的一部話題電影，因為內容是講一些維護公民權利的東西，所以這部電影的資料被某國的網站刪除了，但是這樣反而讓提起更多人看這部電影的興趣。

很多評論都說這是一部很感人的戲，個人看完之後同意，因為故事由一開始講計程車司機男主角只是為了貪錢去拉了一個老外記者的車，但是後來去到目的地之後，被一些正在抗議的市民和大學生所感動，於是主角決定即使不要錢也要出手幫忙，到最後老外記者終於成功脫險，把件事製作成新聞報導。

所以，好看的電影不需要有驚天地泣鬼神的結局，但是如果觀眾看到角色的想法有作出改變，甚至最後犧牲自己去成全大局，至少就可以讓觀眾有所感動。

異形魔怪 5：血脈 Tremors 5: Bloodline (2015)

via 異形魔怪 5：血脈 (豆瓣)

其實我也忘記了為什麼會有這部電影，可能是之前因為看到標題很吸引所以不小心收藏起來。

個人之前沒有看過這個系列的作品，以為已經出到第 5 集，質素應該會有些保證。

結果發現這部《異形魔怪5》原來是一部 cult 片，好不好？看它的評分（豆瓣 4.9）就知道，不過我很佩服老外，就算是部一部 cult 片，也把怪物弄得很丑很討厭，角色被殺時噴血量十分充足，喜歡 cult 片的朋友真的不要錯過。

黑魔塔 The Dark Tower (2017)

via 黑暗塔 (豆瓣)

這部主要是衝著男演員 Matthew McConaughey 和史蒂芬·金的名氣去看，結果讓我有點失望。

原先看《黑暗塔》的簡介還算合理，講異次元世界的東西，應該是一張安全牌，因為看完之後覺得會讓我有種一頭霧水的感覺，我搞不懂主角是有超能力的小孩還是那個黑人槍手，不過也有可能是電影太過高深，所以我們這些凡人才看不懂囉。老實說看完整部戲，我只記得那個異世界的女角色（金秀賢 飾）而已 XD

只有我不在的街道 僕だけがいない街 (2016)

via 只有我不在的街道 (豆瓣)

這又是一部試圖改變時空命運的電影，個人對這類電影本身沒有什麼好感，因為編劇既然可以隨便改變過去和未來，變相就是開外掛，結局隨便編劇要怎樣安排也行，很多時候亂掰一個結局就草草了事。

不過這類時空電影，間中都會有不俗的作品出現，例如《回到最爱的一天》（2015）就是好很好的例子，只是這部的重點不是落在改變結局，反而是希望我們好好活在當下，所以反過來《回到最爱的一天》評分很高，另外還有一部《蝴蝶效應》（2004）都很不錯。

說回來這部《只有我不在的街道》，故事是打著拯救媽媽、救同學的角度出發，所以需要回到過去，看起來很有愛、很偉大，但這樣的故事格局就會較小，成為了一部小品電影，好不好看就見人見智囉！不過個人就覺得電影的節奏比較慢，看得比較累， 是說這是一部推理懸疑電影，不要用愛情片的方式去拍。
