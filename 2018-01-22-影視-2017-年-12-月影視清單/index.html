2017 年 12 月影視清單，個人觀影後的排名如下：

  1. 我的家裡空無一物 わたしのウチには、なんにもない。 (2016)
  2. 星球大戰8：最後的絕地武士 Star Wars: The Last Jedi (2017)

保鏢救殺手 The Hitman’s Bodyguard (2017)

via 王牌保鏢 The Hitman’s Bodyguard (2017)

是部廣受好評的動作電影，應該是因為劇情有趣，加上動作節奏明快，完全爽片無誤，娛樂性十足，值得一看。尤其是中後段摩托車加上快艇的追截戲，好精彩！

《保鏢救殺手》裡面有個比較有趣的問題：到底是保護壞人的保鏢是惡魔鬼，還是刺殺壞人的殺手是惡魔？

我還是覺得要常常刻意去區分好人壞人不是很累嗎？既然大家目標一致，利益相同就一起去幹大事囉！反正很多時候你自己認為是對的事，在別人眼中可能是錯的。

校對女孩河野悅子 2017特別篇 (2017)

via 校對女孩河野悅子 2017特別篇 地味にスゴイ！DX 校閲ガール・河野悅子 (2017)

這部《校對女孩河野悅子》在連播的時候已經有在追看，當然是喜歡，縱使劇情不算是特別的提材，但是收視依然十分之好，我想主要原因是歸功於女主角石原里美的個人魅力啦！

這次電影加入了木村佳乃。有點像是呼應了日劇《First Class 2》的感覺，因為那部日劇也是講雜誌的故事。當然《First Class》是講女人之間勾心鬥角的故事，跟這部《校對女孩》差好多。

說到底《校對女孩》仍然是那種典型的「純愛萬歲」日劇，偶爾看看是會令人蠻開心的，尤其是能夠可以看到可愛的石原里美啦 XD

弦途有你 Walk the Line (2005)

via 與歌同行 Walk the Line (2005)

又來清一下放了好久的電影。

這部應該是因為女主角 Reese Witherspoon 拿了奧斯卡女主角才收藏起來的，個人覺得這部 《Walk the Line 》不是那種高潮迭起、超級精彩的電影，反而是適合喜歡 60、70 年代美國鄉土音樂朋友去看。

《Walk the Line 》故事是改編自美國著名歌手 Johnny Cash 的真人真事。看電影完會覺得，能夠有一個賢慧的女朋友/老婆真的很重要，因為她可以在你困擾的時候拉你一把，幫你打打氣。

王牌特工2：黃金圈 Kingsman: The Golden Circle (2017)

via 王牌特工2：黃金圈 Kingsman: The Golden Circle (2017)

《Kingsman》是一部近年很有話題的電影，因為裡面用了很多生動有趣的拍攝手法，所以在 2015 年第一集 《Kingsman: The Secret Service》 上映之後很快就推出了第二集。

這集整體來說同樣的精彩，前半部和中段都有被劇情有趣的設定驚喜到，相比之下，後段的收尾和結局有點遜色，因為作為大 BOSS 的 Poppy 根本不能打呀 Orz 甚至是最後主角跟 Whiskey 的打斗也是一下就搞定了，太弱了吧 XD

星球大戰8：最後的絕地武士 Star Wars: The Last Jedi (2017)

via 星球大戰8：最後的絕地武士 Star Wars: The Last Jedi (2017)

老實說個人不是《Star Wars》的粉絲，但是作為電影愛好者，《Star Wars》絕對是一部必看的作品，如果有時間的話更是建議把之前的七集都拿出來看一下。

電影系列來到第八集，覺得這部跟之前最大的不同是多了角色犧牲，多了感動，劇情的安排更加緊湊，高潮一浪接一浪，因為在之前看到電影得到了許多好評，所以我也特定跑去了電影院看這部戲，那個音效體驗真的很棒，令人更期待下集的來到。

我的家裡空無一物 わたしのウチには、なんにもない。 (2016)

via 我的家裡空無一物 わたしのウチには、なんにもない。 (2016)

這部日劇蠻特別的，是教人把家裡多餘東西丟掉的日劇，我是因為看到一篇文章《我的家裡空無一物》：把你的家變成斷舍離實驗室吧！ - 每日頭條 才知道，整部日劇只有六集，很快就可以追完，之後會忍不住跟女主角一樣想把家裡多餘的東西通通清掉 XD

《我的家裡空無一物》女主角是因為失戀而變成丟東西狂魔，當然有時候也會遭到身邊的伴侶和家人誤解，所以有些東西在丟之前還是要問一下家人的意見會比較好啦！

日劇有講到一個丟東西不錯的方法，當我們有件物品很想丟掉但是又猶豫不決的時候，可以先跟這個物品保持一段距離，如果隔了一段時間發現真的沒有再碰了，證明我們對這個物品沒有需要，之後就可以賣掉或丟掉。

日劇的最後一集很有意思，女主角原先是問媽媽死後想要留下些什麼，但是卻被媽媽反問，女兒想要媽媽死後留下什麼給她。的確，對於死去的人來說有什麼留下來是沒有意思的，重點是活著的人呀。

這部日劇讓我想起之前那部電影《Up in the Air》（2009），一個人真正需要用到的東西沒有很多，多餘的物品只是會增加負擔，阻礙你前進的步伐，勇敢果斷地捨棄，你發現得到的會是更多。

滾蛋吧！腫瘤君 (2015)

via 滾蛋吧！腫瘤君 (2015)

這部完全是因為看到豆瓣有 7.4 分才會拿來看看，發現只是部蠻可愛的電影，可愛是指電影用了大量二次元的風格去創作，風趣幽默，再配合女主角患上癌症的劇情，可以說是笑中有淚。

覺得找吳彦祖當男主真是不太搭，吳彦祖是是帥，但是個人就認為他不適合濱這種這種角色，反而是他當年演的《門徒》（2007）跟《新宿事件》（2009）比較讓我深刻，無辦法啦！人長的帥，角色就是會容易被定型。
