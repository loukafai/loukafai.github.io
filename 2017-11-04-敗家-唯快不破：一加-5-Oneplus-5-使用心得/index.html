使用了這台手機已經有幾個月了，是時候分享一下心得。（首圖是借用微博網友 @翻滾吧Kratos 的圖片，拍得真好）

個人是由之前的榮耀 v8 換成這台一加5，最大的差別是手機的指紋識別由機身背面改回機身前面，再來就是接近原生 Android 的 氧 OS 系統，整部手機在使用上十分爽快順手。

至於一加5 的開箱文就不拍了，因為以下兩篇的網友已經寫得很棒了，而目前一加5 已經接近缺貨，所以這篇就當作一個純分享，即使要買，接下也會直接選一加 5T 了吧！感覺 5T 也快將要推出了（點頭）

先說結論

1、喜歡一加 5 的點

  1. 3.5mm 耳機孔的保留，隨手拿起一支耳機都可以用，邊充電邊聽歌也可以
  2. 氧 OS（oxygenOS）有接近原生 Android 系統的體驗，而且可自訂性高
  3. Snapdragon 835 + 6GB Ram 的高效能
  4. 相機啟動和對焦速度都很快
  5. 機身左設有獨立的三段式音量開關鍵
  6. 「Dash 閃充」充電真的好快

2、不喜歡一加 5 但仍可以接受的點

  1. 相機沒有光學防手震，低光源下的成像稍弱
  2. 螢幕有輕微偏紅，不過看久了會習慣
  3. 螢幕有所謂的果凍屏現像，絕對不影響正常使用
  4. 氧 OS （OxygenOS）可自訂性高但功能仍不足夠（例如單手模式、應用雙開）

3、前置指紋識別才好用？

（ 圖片來自：一加官網）

在經過使用了一年後置指紋識別的榮耀 v8，覺得指紋識別放在機身背後也還算方便，尤其對於會把手機放在褲子口袋的朋友就更加的實用，因為把手機從口袋掏出來時食指就已經可以碰到解鎖位置，對於大尺寸的手機也可以拿得很穩。

相反如果是前置指紋識別的話還要把姆指拉一下，多開幾次鎖之後姆指會很累，而且手機也比較拿不穩，但是手機平放在桌面使用上就可以直接用手指解鎖，不用再拿起手機，特別是充電的時候。

4、特有的三段式開關設計

（ 圖片來自：一加官網）

「三段式開關設計」是一加系列手機一直都有的功能，是目前 Android 手機中唯一有這種設計的品牌。「三段式開關設計」在使用上有點類似 iPhone 上的靜音開關，但是它就多了一段是「勿擾」模式的選擇，即是在這個模式下只會接收特地聯絡人的來電，聽起來很不錯！不過我一般都只會直接設靜音比較快。

總的來說覺得「三段式開關設計」是個很好的設計，因為可以不用打開螢幕就可以直接調靜音，對於經常需要在兩種模式之間切換的用家真的很方便，希望這個設計可以一直保留。

5、氧 OS - 接近原生 Android 系統的魅力

（ 圖片來自：一加官網）

我的一加5 是直接在某東網上購買的，而在中國內地直接買到的一加5 預設都是使用氫 OS，不過沒關係，只要去 Oneplus 官網下載個氧 OS （OxygenOS） 丟進手機，之後進入手機裡的「設定」裡面點「手機更新」就可以簡單的由氫 OS 刷成氧 OS 囉！

下面就有 mobile01 網友的分享。

  • Android其他品牌硬體綜合 - ONEPLUS5 氫氧互刷 - 手機討論區 - Mobile01

把一加5 刷成 氧 OS 之後就馬上可以使用 Google Play、Gmail、Google Map 一系列 google 服務，對港澳臺用家來說十分方便。

氧 OS 的設計十分接近原生 Android 系統，再配搭 Snapdragon 835 + 6GB Ram 運行起來超級流暢，是在使用 Android 系統的經驗裡前所未有的體驗，值得一試！

不過因為之前習慣了大陸國產手機的「全家桶」，這回用氧 OS 真是不習慣，許多常用的功能例如單手模式、應用雙開、懸浮球、手機管理等都一一欠奉，只希望之後原廠將更多功能慢慢補強，不要浪費這麼強大的硬件。

氧 OS 除了有方便的「體感手勢」（雙擊亮屏、黑屏手勢）不用說之外，還能夠對「按鍵」的功能自訂義，這是很多其它品牌的 Android 系統都沒有的。

6、「Dash 閃充」真的很快，但沒有 QC3.0 實在不便

（ 圖片來自：一加官網）

在使用過一段日子之後，發現「Dash 閃充」真的非常、非常、非常快，但是一加5 選擇了「Dash 閃充」卻放棄了 QC3.0，對於一般使用者來說是有影響的，因為支援「Dash 閃充」是少眾，相反 QC3.0 才是市場主流，而且市面上根本沒有支援「Dash 閃充」的移動電源，對比 QC3.0 隨便找都一大堆，對於有使用移動電源快充的用家會比較不方便。

不過，在網路上看到分享，是可以上某寶找「qc 轉 dash」的一個工具，使一些支援 QC2.0 / QC3.0 的移動電源支援「Dash 閃充」。

7、低調支持生活防潑水

雖然一加5 在手機官網的參數頁面沒有標明，但是在一些網友的實測和拆機視頻中可以看到一加5 是有基的防水功能，只是不敢明講出來，而我個人也曾經在使用手機的情況下淋到雨，最後手機也是完好，所以認為一加5 是能夠應付基本的防潑水。

  • 一加手机5拆机：拆完扔水里依旧能防水!

8、2999RMB 的性價比？

個人覺得，2017 年到目前為止使用 Snapdragon 835 最具性價比的手機除了一加 5 之外就是售價為 2499RMB 的小米6 了。其實兩台手機的售價差距不大，所以就看買家想要的是什麼了。

如果像我一樣喜歡大螢幕（5.5 寸）、輕巧的操作系統（接近原生 Android 的氧 OS 系統 ）、3.5mm 耳機孔，不用再猶豫，一定是選一加5 了。

9、續航給力

一加5 是佩備 3300 mAh 電池，得益於 Snapdragon 835 的低功耗和比較省電的 AMOLED 螢幕，個人是足夠中度使用一天，不輸我之前的 榮耀 v8（3500 mAh 電池）。

剛換這台一加 5 的時候因為怕這顆 3300 mAh 電池不給力，所以而且系統沒有內置額外的電源管理 App （只有 Android 7.1 內建的），所以就會配搭「黑域」去使用，然後真的十分省電，中度使用一天也用不完。後來因為沒有 ROOT 過的手機用「黑域」每次重開機都要用電腦啟動，這點比較不便，所以就沒有再用了，而且手機的續航力能維持住，之後真的不行再用「黑域」吧。

10、雙攝成像效果不俗

（ 圖片來自：一加官網）

相機啟動快、對焦迅速是一加5 的優點，再加上今年流行的大光圈人像模式，讓小小的手機可以拍出單反相機的效果，所以我對一加5 的相機很滿意。
