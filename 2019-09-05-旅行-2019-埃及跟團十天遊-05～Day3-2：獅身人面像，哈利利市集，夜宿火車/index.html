[旅行] 2019 埃及跟團十天遊 05～Day3-2：獅身人面像，哈利利市集，夜宿火車

2019-09-05
2019-egypt-10days, 2019-埃及, egypt, http://schemas.google.com/blogger/2008/kind#post, lift, trip, 埃及, 旅行, 生活, 非洲

繼續【2019 埃及跟團十天遊】之旅。在拍完三個金字塔之後，來到今天的另一個重頭戲，獅身人面像。

這個獅身人面像由建造到發現都十分傳奇，這次來到除了拍照，當然要聽聽關於它的故事。

第三天行程

飯店早餐：Grand Nile Tower → 吉薩金字塔 → 金字塔最佳拍照點 → 獅身人面像 → 午餐：絲綢之路 → 哈利利市集 → 旅館盥洗 → 吉薩火車站 → 夜宿火車

▲ 導遊在車上先講解一下獅身人面像 ( via 20190402 出發去看獅身人面像 - YouTube)

▲ 穿過小販區

▲ 來到入口神廟

獅身人面像

獅身人面像或稱人面獅身像，是一座位在卡夫拉金字塔旁的雕像，外型是一個獅子的身軀和人的頭。獅身人面像長約73.5公尺（241英尺），寬約19.3公尺（63英尺）和高約20.22公尺（66.34英尺）。獅身人面像是現今已知最古老的紀念雕像，一般相信是在法老卡夫拉統治期間內建成。 ( via 獅身人面像 - 維基百科，自由的百科全書)

▲ 導遊介紹獅身人面像，原來蓋神廟的石頭也是一個奇蹟 ( via 20190402 獅身人面像介紹 - YouTube)

▲ 一位熱情的中國大媽主動搶我們手機來幫拍照

▲ 人山人海，不過越往裡面走，人越少

▲ 抓到空檔，正妹長腿照一張，收工

午餐：絲綢之路

▲ 中餐館，旅行團都會照顧愛吃中餐的團友來安排一個中餐，不過我個人比較愛吃在地美食，重點是埃及的中餐一定沒有家鄉的好吃呀

▲ 菜色好多，都以蔬菜為主，大家一定能食飽囉

▲ 埃及的柳丁香甜多汁，難怪很多都出口到國外

▲ 捕獲偉士牌一台

哈利利市集

哈利利市集 Khan el-Khalili，這座大市集就坐落在歷史悠久的市中心，宛如迷宮般的窄巷，這裡有著各式各樣的異國商品和紀念品讓您挑選，保證讓您逛的眼花撩亂，別忘記這裡殺價是一定要的喔！ ( via 雄獅旅遊)

▲ 我們只有在邊待一個多小時，所以要抓緊機會購物和拍照。而其實很多伴手禮在其它地方也買到，不用緊張

▲ 我們在這家綠綠的咖啡店集合，不想逛的人可以在這喝咖啡和抽水煙 XD 我們的話當然要去逛去拍照囉

▲ 哈利利市集真是充滿異國風味，很好拍

▲ 香精瓶

▲ 我們在哈利利市集的戰利物就是小小冰箱貼，10 Le 一個，老闆面超臭都不能殺價

▲ 回到咖啡店集合，點一杯導遊推薦的芒果汁，聽說完全沒有加水，果然超級香甜

▲ 我這張好瘦 XD

▲ 司機開一台大大的旅遊巴士在市集中穿梭，超厲害

旅館盥洗

▲ Amarante Pyramids 飯店，今天晚上睡火車，所以安排我們先去一定飯店洗澡，4 個人一間

▲ 我們讓同行的情侶先洗，因為難得有機會，我們就先跑去路口的咖啡店買杯在地的咖啡品嚐

▲ 連雞蛋仔都可以吃到

▲ 點了一杯拿鐵 18 Le

▲ 然後經過一家小市多，補了一點零食給晚上火車吃

▲ 買到有名的新娘紅茶，8 egp

▲ 覺得剛剛的拿鐵沒有特色，之後又跑回來再點一杯土耳其咖啡，這個喝完有渣渣在杯底才是中東咖啡特色的味道呀

▲ 聽導遊介紹埃及的火車，然後埃及人的開車態度是多麼的隨性，開車撞到別人都不用擔心 ( via 20190402 出發去火車站 - YouTube)

吉薩火車站

▲ 晚上 6：50 到達吉薩火車站

▲ 行李不用擔心，有付小費給小弟給我們直接送到火車上

▲ 經過安檢

▲ 塔火車前可以補些零食

▲ 然後大家就在月台上等火車，順便觀察一下埃及人的生活

夜宿火車

▲ 時間是 7:43 ，等了半個多小時，火車終於來到，心得果然真是：火車有到就好 XD

▲ 車廂原先的樣子，前面有張小餐桌可以拉出來食晚餐，然後晚點服務員會進來把椅子變成上下兩張床

▲ 變身後

▲ 聽說是旅行社加碼的水果

▲ 火車餐盒，有牛肉飯，蔬菜，麵包，埃及必吃柳丁一個，味道不要太期待，反正有東西吃就很不錯了

▲ 房間內有洗手盆

▲ 在火車上要用轉接頭才能用電

▲ 冷氣可以調，不過調到最低一樣好冷

▲ 每節車廂共用一個廁所，看到馬桶底下是亮亮的代表東西直接噴到鐵軌上，而且廁所的衛生程度………..

▲ 晚上沒有什麼事情可以做，早點睡覺吧
