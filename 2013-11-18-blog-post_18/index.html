▲ 澳門一日遊～～不專業地陪

學長是我在台灣求學時期對我影響最多的人之一，而且我去年的新加坡之旅也有去拜會他老人家一下，他還請了我們吃了好吃的沙嗲，最近他剛好路過澳門找我，一定要好好招呼他一下。

一天逛完澳門

行程如下：

早餐：南屏雅敘 → 大三吧 → 炮台 → 新馬路 → 點心：義順牛奶、瑪嘉烈葡撻 → 何東圖書館 → 風順堂區 聖老楞佐堂 → 主教山教堂 → 媽閣廟 → 午餐：媽閣銘記澳葡美食 → 氹仔賭場區 → 龍環葡韻博物館 → 氹仔官也街 → 大利來豬排包 → 路環市區 → 晚餐：長城燒烤 → 澳門旅遊塔


行程有沒有很豐富？學長說要看有葡國特色的建築風格，這些景點都超有葡國 fu 的，所以，鄭家大屋、觀音堂那些中國風的景點就可以忽略了。

原則上要一天逛完澳門是沒問題的，但重點是要有交通工具，而我們的交通工具就是我的小小機車，NEX 125cc 很久都沒有載過人了，就操它一下吧！有機車可以一天內逛完澳門，坐巴士和走路的朋友就要預算 1 天半才夠。

P.S 以下傷眼的照片都是用 Samsung S3 所拍 (有幾張是 iPhone 4S)，畫質沒有很好，加上當天是陰天，畫質更加差上加差，不過好像 4S 的畫質比 S3 還要好，我要換手機！！

▲ 南屏雅敘。先來個有澳門特色的茶餐廳風味早餐，南屏文治 (其實我也是第一次來 XD)，還好學長的反應不錯，總算保得住澳門美食的面子。

▲ 澳門最重要的地標，大三巴排坊（這樣是有比較帥嗎？）學長看到時跟許多人的反應一樣：就這樣子！？
